はチェックアウトされていません…バンドルをインストールしてもヘルプは修正されません。

概要

https://github.com/intridea/omniauth.git (at master) is not checked out. Please run `bundle install` (Bundler::GitError)

それで、私は何をすればいいでしょうか？バンドルインストールは開発では機能しますが、実稼働サーバーにプッシュしてデプロイするときは機能します。運用サーバーでバンドル インストールを実行した後でも、このエラーが発生します。

解決策

おそらくPassengerを実行していると思われます。これは一部のソリューションの問題です - http://code.google.com/p/phusion-passenger/issues/detail?id=505

Bundle install –deployment を実行してみてください
