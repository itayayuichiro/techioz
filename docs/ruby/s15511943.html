RVM と OpenSSL に関する問題

概要

同僚のために新しい MacBook をセットアップしようとしています。うまくいかない。

まず、OpenSSL をインストールします。

    rvm reinstall all --force

You must recompile Ruby with OpenSSL support or change the sources in your Gemfile from 'https' to 'http'. Instructions for

ところで、gem ファイルには「source https://rubygems.org」と書かれています。

アイデアは？

解決策

OPは実際に私の一日を救ってくれました！ 私の場合：

$ rvm pkg install openssl
$ rvm remove 2.4
$ rvm install 2.4 --with-openssl-dir=$HOME/.rvm/usr
$ gem install bundler
