Ruby gem コマンド クラッシュ シェル Fastlane と Cocoapods

概要

ここ 1 か月間、fastlane と Cocoapod の実行に問題が発生しています。何が問題なのかわかりますか？ RVMを試したり、バージョンを変更したりしましたが、何も機能しませんでした。

fastlaneを再インストールすると、cocopodaが再び動作し始めました。現在、fastlane が機能しません。以下のエラーが発生します。

解決策

ファストレーンをどのように呼び出しているのかわかりません。つまり、fastlane -v ではなく fastlane –v を実行している可能性があります。

$ fastlane --help
...
-v, --version        Display version information
