ファイル名内のrubyzipおよびunicode文字

概要

Rubyzip gem と Zip::ZipOutputStream クラスを使用して zip アーカイブを作成していますが、Unicode 文字 (キリル文字) で問題が発生しました。アーカイブでは、????? のような疑問符として表示されます。 ???? ??.doc。 Rubyzip は Unicode をサポートしていますか?

解決策

Rubyzip のメソッドを調べてみましたが、rubyzip では言語を変更できないようです。おそらく、コンピュータのデフォルトのコードページが使用されます。 chilkat で対処できない特定の要件がない限り、この例のように代わりに chilkat zip を使用できます。
