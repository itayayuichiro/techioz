Cocoapod のインストール中のエラー

概要

最新の CocoaPods (現在 1.0.1) をインストールしようとしていますが、次のエラーが表示されます。

$ sudo gem install cocoapods
Password:
Fetching: i18n-0.7.0.gem (100%)
Successfully installed i18n-0.7.0
Fetching: thread_safe-0.3.5.gem (100%)
Successfully installed thread_safe-0.3.5
Fetching: tzinfo-1.2.2.gem (100%)
Successfully installed tzinfo-1.2.2
Fetching: minitest-5.9.0.gem (100%)
Successfully installed minitest-5.9.0
Fetching: concurrent-ruby-1.0.2.gem (100%)
Successfully installed concurrent-ruby-1.0.2
Fetching: activesupport-5.0.0.gem (100%)
ERROR:  Error installing cocoapods:
activesupport requires Ruby version >= 2.2.2.

インストールガイドにはこう書かれています

Ruby は私の好みの言語ではないので、cocoapods gem をインストールする方法を誰か手伝ってくれませんか?

OSXバージョン：OS X El Capitan 10.11.5

解決策

まず、古いバージョンの「activesupport」(Ruby 2.0 と Cocoapod の両方と互換性がある) をインストールしてから、Cocoapod をインストールします。 sudo gem install activesupport -v 4.2.6
