ターミナルから coocapod をインストールするときにエラーが発生しました

概要

ターミナルから Cocoapod をインストールする 「エラー: Cocoapod のインストール中にエラーが発生しました: Ruby および RubyGems と互換性のある Cocoapods-downloader (>= 2.0) のバージョンはありません。探している gem の古いバージョンをインストールしてみてはいかがでしょうか? Cocoapods-downloader には Ruby バージョン 2.7.4 以上が必要です。現在の Ruby のバージョンは 2.6.10.210 です。」

M1 Airを使用しているのですが、それが関係しているのでしょうか？

解決策

おそらく、macOS に組み込まれている Ruby バージョンを使用していると思われます。これは古くからあるものであり、要件が異なる複数の Ruby プロジェクトがある場合は簡単に管理できないため、これは災難の元です。

rbenv をインストールし、それを使用して新しいバージョンの Ruby をインストールし、それを使用して Cocoapod をインストールします。
