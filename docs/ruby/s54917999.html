VSCode を使用して Ruby スクリプトを実行するにはどうすればよいですか?

概要

私は MacOS を使用しており、VSCode、Ruby、rubocop、Ruby Solargraph、およびそれらの gem の依存関係をインストールしました。

これを完了したら、VSCode 内のターミナル パネルを使用せずに Ruby スクリプトを実行するにはどうすればよいでしょうか?

解決策

コードランナー拡張機能を使用できます。

拡張機能を追加したら、Command + Shift + Pを押して言語ごとに実行を選択し、Rubyを選択します
