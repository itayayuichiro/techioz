Rails アクティブ ストレージ - リンクを開く/ダウンロード

概要

Active Storage に関する質問: Rails 5.2 に更新したばかりで、BLOB として保存されている添付ドキュメントを開こうとしています。現在、root_path にリダイレクトされていますが、代わりに開く/ダウンロードする方法はありますか?

Rails ビューでの私のコードは次のとおりです。

<%= link_to(document.filename, Rails_blob_path(document, disposition: “attachment”)) %>

解決策

ダウンロードする：

<%= link_to document.filename, rails_blob_path(document, disposition: 'attachment') %>

プレビューするには:

<%= link_to document.filename, rails_blob_path(document, disposition: 'preview') %>

ソース - ActiveStorage#ファイルへのリンクのドキュメント
