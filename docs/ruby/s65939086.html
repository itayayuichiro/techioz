特定のバンドラー バージョンをインストールする方法

概要

Ruby を初めて使用します。 Ruby 2.7.2 をインストールしましたが、特定のバンドラー バージョンをインストールしたいと考えています。実行するとき:

gem cleanup
 gem cleanup bundler
 gem list bundler
 bundle install 2.2.5

次のエラーが発生します:

Cleaning up installed gems...
Clean up complete
bundler (default: 2.1.4)
ERROR: "bundle install" was called with arguments ["2.2.5"]
Usage: "bundle install [OPTIONS]"

特定のバンドラー バージョンをインストールするにはどうすればよいですか?

解決策

あなたがやろうとしていることは、次のコマンドで達成できると思います。

gem install bundler -v 2.2.5

の代わりに

bundle install 2.2.5
