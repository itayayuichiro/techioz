RubyのXMLの差分? [閉まっている]

概要

2 つの XML ドキュメントを Ruby でマージする最良/最速の方法は何ですか?

2 つの XML ファイルがあります。1 つは視覚的に魅力的なようにフォーマットされており、もう 1 つはそうではない (コメントと空白が削除されている) もので、全体の一部のノードにいくつかの変更があり、頻繁に変更されます。そこで、何が変更されたのかを確認し（すべてに ID があるわけではない可能性があります）、古い文書を書式設定された文書とマージするための、シンプルで効率的な解決策を見つけようとしています。
