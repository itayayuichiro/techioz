M2 Mac で初期 React Native テンプレートを使用した iOS バージョンを実行すると問題が発生する

概要

私は最近 Mac で React Native を使い始めました。公式ドキュメントに従ってすべてをセットアップしました。次に、npx react-native init ProjectName を使用して新しいプロジェクトを作成します。すべてエラーなしで動作しますが、iOS 上で実行しようとすると (npm run ios を使用して)、以下の 2 つのエラーのみで失敗します。

使用環境に関する情報:

追伸: React Native で作成された別のプロジェクトは、まったく同じ構成で魔法のように機能し、何も変更されていませんでした。 (プロジェクト間でエラーが矛盾しているようです)

よろしくお願いします！

インターネット上でこのエラーに関連するすべてのことを試しましたが、何も機能しませんでした。私の問題は非常に特殊なようで、問題を理解できません。

解決策

この質問を見つけた人を助けたいと思ったのですが、問題はこのアプリの直接の親フォルダーの名前にスペースが含まれていることであることがわかりました。私の意見では奇妙な問題ですが、それは修正されます。 これが誰かの役に立てば幸いです!
