現時点では、M1 プロセッサーに特化したものを何も作成する必要がないことに注意してください。デフォルトの arm64 ターミナルからすべてを通常どおりインストールしました。

概要

ポッドのインストール中に「find_spec_for_exe」: can’t find gem Cocoapods エラーが発生します。 Stackoverflow で解決策を試しましたが、修正できません。どうすれば解決できますか?

Ruby には 2 つの異なるバージョンがあるようです。私の問題はこの状況に関係していますか?

解決策

Macを再起動します

Run sudo gem install cocoapods

それから

sudo gem update --system
sudo gem install cocoapods -n/usr/local/bin
