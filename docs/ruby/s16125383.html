iOS 登録済みプロトコル ハンドラー用の API [終了]

概要

登録された iOS プロトコル ハンドラーが存在するかどうかを確認できる API はありますか?私に与えられたいくつかのプロトコル ハンドラーが有効かどうかを確認しようとしています。

解決策

アプリケーションの URL スキームについて話している場合は、次のものを使用できます。

    [[UIApplication sharedApplication] canOpenURL:<#(NSURL *)#>]

URL スキームを開くことができるアプリがある場合は YES を返し、存在しない場合は NO を返します。
