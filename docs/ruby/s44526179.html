解決策

将来この問題に遭遇した場合は、次のように、異なる rspec gem の複数のバージョンがインストールされているかどうかを確認してください。

gem list | grep rspec

そうした場合は、Gemfile.lock にあるものと一致しない gem のバージョンを次のように削除してください。

gem uninstall <gem_name> <gem_version>

これらの手順に従った後、rspec spec を正常に実行することができました。
