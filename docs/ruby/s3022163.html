Ruby on Railsで月の名前を取得するにはどうすればよいですか?

概要

だから私は私の見解で作成します:

<%=date=Date.today%>

日付から月の名前を取得するにはどうすればよいですか?私はsthのようなことをしようとしていた

 <%= DATE::ABBR_MONTHNAMES(date.month)%>

しかし、成功しませんでした。エラーが発生し続けます: 初期化されていない定数 ActionView::Base::CompiledTemplates::MONTHNAMES

定数を初期化するにはどうすればよいですか、それとも日付形式から名前を取得する他の方法はありますか?

ご回答いただければ幸いです。

解決策

これを参照してください

<% @date = Date.today  %>  
<%= @date.strftime("%B")%>

もし

@date >> Fri, 11 Jun 2010

それから

@date.strftime("%B") >> "June"
