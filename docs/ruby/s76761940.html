/index.html を含まないページ URL を表示するにはどうすればよいですか?

概要

私はジキルとリキッドを初めて使用するのですが、canocial URL に問題があります。投稿 (記事) ページでは、すべてが完璧に機能します。ただし、カテゴリーページでは、URL に /index.html が追加されます。

カテゴリページのHTMLコードはこんな感じです

`{{site.url}} / {{page.url}}`в _config.yml, как это:`Постоянная ссылка: '/: категории /: название'`в браузере в строке url он будет отображаться без /index.html. Все как нужно. Например:`https://site.ru/blog/otzyvy/`Но в коде страницы это отображается так:`https://site.ru/blog/otzyvy/index.html`Как это можно исправить?

解決策

ページにパーマリンク設定を追加し、パスの末尾にスラッシュが含まれていることを確認する必要があります。最後のスラッシュを削除すると、index.html との相対パスが作成されます。
