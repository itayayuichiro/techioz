Twilio gem をインストールできない - libxml の問題

概要

通常のスクリプトと Rails アプリの両方に twill-ruby gem をインストールしようとして何時間も苦労した後、私はプロにたどり着きました。

表示されるエラーは「Gemfile 内:」です。 twilio-ruby は 5.2.0 に解決されました。 libxml-ルビー 」

libxml gem を gem ファイルにインストールしようとしましたが、「Gem::Ext::BuildError: ERROR: Failed to build gem Native extension」というエラーが表示されて失敗します。そして一番下に「libxml-ruby (3.0.0) のインストール中にエラーが発生したため、Bundler は続行できません。」と表示されます。 バンドルする前に gem install libxml-ruby -v ‘3.0.0’ が成功することを確認してください。」

私が使用している Rails のバージョンは 4.2.4 で、OS は MACOS Sierra です。 twilio gem をバンドルインストールしようとしたときの出力全体は以下のとおりです。 Stack に関する記事をいくつか読みましたが、これまでのところ何も役に立ちませんでした。

解決策

解決策を見つけました！ (Debian 9 で実行)

sudo apt-get install -y libxml2-dev

gem install libxml-ruby -v '3.0.0' 

bundle install

libxml2を使用しようとするlibxml-rubyに問題があるようです
