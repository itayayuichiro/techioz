RVM Ruby 許可が拒否されました

概要

マルチユーザー用にrvmをインストールしましたが、

バンドルインストールを実行するとエラーが発生します。

linux:/var/rails_apps$ bundle install
ERROR: RVM Ruby not used, run `rvm use ruby` first.        

linux:/var/rails_apps$ rvm use ruby
mkdir: cannot create directory `/usr/local/rvm/log/ruby-2.0.0-p247': Permission denied

私はすでにやった、

sudo chmod 777 /var/rails_apps/

提案してください..

解決策

RVM をマルチユーザーでインストールしていますが、(あなたの場合と同様に) 問題が発生する可能性があるため、これはお勧めできません。

すでにそこにいるので、これで修正されるはずです。

rvmsudo rvm get stable --auto-dotfiles
rvm fix-permissions system

また、rvm グループに所属していることを確認してください。そうでない場合は、自分自身を追加します。

rvm group add rvm $USER

新しいシェルにログインします (ログアウトしてから再度ログインします)。
