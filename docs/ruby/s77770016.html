JavaScriptに変更がある場合、rails importmap-railsに自動コンパイラはありますか? (webpacker や esbuild のように)

概要

デフォルトで importmap を持つ新しい Rails プロジェクトがあります。

変更を更新するには、毎回次のように入力する必要があります

rails assets:precompile

しかし、webpacker では変更を反映して再コンパイルされることはわかっています。 importmapsでも同じことを行うことは可能ですか?

1 つのパッケージをインポートし、スティミュラスを使用して小さな JavaScript コントローラーを実行するだけです

解決策

@Alex の言うこと -

rake assets:clobber     # Remove compiled assets

そのコマンドは私を助けてくれます、ありがとうアレックス！
