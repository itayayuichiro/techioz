rbenv で利用可能な Ruby のバージョンが表示されない

概要

vagrant マシンに rbenv をインストールしましたが、利用可能なすべての Ruby バージョン rbenv バージョンをリストしようとすると、次の出力が表示されます。

システム (/home/vagrant/.rbenv/version で設定)

これは何なのか誰か知っていますか？

解決策

そのコマンドの出力が何を意味するのか知りたいのですか? rbenv のドキュメントを見ているところ、rbenv は Ruby の 1 つのバージョン (システムにインストールされているバージョン) しか認識していないことが示されていると思います。

編集: それがあなたが探していた答えではない場合、何を期待していたか教えていただけますか?そのマシンに他のバージョンがインストールされていましたか?

編集2: マシンにインストールできるがまだインストールされていないバージョンのリストを表示したい場合は、次を実行します。

rbenv install --list

このコマンドの詳細なドキュメントは次のとおりです。これは役に立ちます!
