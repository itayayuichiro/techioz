概要

  def show
    @post = Post.find(params[:id])
    respond_to do |format|
      format.html # show.html.erb
      format.json { render json: @post }
    end
  end

解決策

昨日このエラーが発生したので、Ruby 2.4 と Rails 3.2 で動作するようにする回避策を作成しました。これもruby 2.5.9p229およびRails 3.2.22.5で動作することが確認されています。

私がアイデアを得たコミットは次の場所にあります。 https://translate.google.com/translate?hl=ja&sl=en&tl=ja&u=https://github.com/rails/arel/commit/dc85a6e9c74942945ad696f5da4d82490a85b865

Rails 初期化子に以下を含めるだけです。

module Arel
  module Visitors
    class DepthFirst < Arel::Visitors::Visitor
      alias :visit_Integer :terminal
    end

    class Dot < Arel::Visitors::Visitor
      alias :visit_Integer :visit_String
    end

    class ToSql < Arel::Visitors::Visitor
      alias :visit_Integer :literal
    end
  end
end
