RubyMine で Ruby SDK を選択できない

概要

SDKの選択

適用ボタンは何も起こりません

2 番目の画像の適用ボタンをクリックしても、何も起こりません。 OK、どちらも機能しません。この問題を解決する方法を知っている人はいますか?

cmd:ruby -v の結果は C:>ruby -v となります。 Ruby 3.2.2 (2023-03-30 リビジョン e51014f9c0) [x64-mingw-ucrt]

解決策

ブランチを切り替えた後、RubyMine はインタプリタとして Ruby SDK を選択できるようになりました。理由はわかりませんが、ブランチを切り替えた後、問題は解決されました。

もう 1 つの提案は、RubyMine を閉じた後に .idea フォルダーを削除することです。 IDE 内でプロジェクトを再度開くと、再作成されます。
