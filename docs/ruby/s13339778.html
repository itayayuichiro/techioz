条件付き配列の最初の要素

概要

いくつかの条件を満たす配列内の最初の要素を見つけるためのこれより短い方法はありますか?

my_array[ my_array.index {|x| x.some_test} ]

解決策

これを試して：

my_array.find{|x| x.some_test }

または、ここにショートカットがあります (思い出させてくれた @LarsHaugseth に感謝します)

my_array.find(&:some_test)
