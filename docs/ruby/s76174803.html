chromedriver.storage.googleapis.com への TCP 接続を開くことができませんでした:443 (実行期限が切れました)

概要

次のエラーが表示されます。 chromedriver.storage.googleapis.com への TCP 接続を開けませんでした:443 (実行期限が切れました)

いくつかの機能仕様の実行中

私のGemfileにはこれがあります:

  gem 'selenium-webdriver'
  gem 'webdrivers'

ローカルに chromeuum がインストールされており、パス上にあります。

ヒントは大歓迎です、thx

解決策

以前にもこのエラーに遭遇したことがあります。この問題を解決するのに役立ったのは、Google Chrome と Chromedriver のバージョンが一致していることを確認することでした (必要に応じて再インストールしました)。

コンテキスト: Google Chrome のバージョンを 114.0.5735.198 から 114.0.5735.90 に変更しようとして、Google Chrome を削除し、最初から再インストールしました。 chromedriver -v はまだ目的のバージョン (114.0.5735.90) を表示していましたが、エラーが表示され始めたのはこの時だったため、パージ中に何かが起こった可能性があります。

rspec テストを実行するとき。 chromedriverを再インストールすると問題が解決しました。
