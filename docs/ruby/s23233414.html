Heroku: Seeds.rb を既存の Rails アプリにプッシュするにはどうすればよいですか?

概要

私はすべてのアプリのデータをローカルの Seeds.rb に保存します。ただし、すべてを Heroku にプッシュすると、アプリは正常に動作しますが、データはありません。大量のデータを再入力したくないのですが、誰か私を助ける方法はありますか?

解決策

heroku run rake db:seed
