erb ビューでハイパースタック コンポーネントをレンダリングするにはどうすればよいですか? [閉まっている]

概要

たとえば、多くのロジックを含む erb ページがあります。 Hyperstack を使用してこのページにユーザー メッセージを作成したいと考えています。この erb ページでハイパースタック コンポーネントをレンダリングするにはどうすればよいですか? 更新: Hyperstack コンポーネントが aplication.erb に埋め込まれているため、これが可能であることはわかっています。

解決策

解決策を見つけました。

erb ファイル:

<%= react_component('TestComponent') %>

ハイパースタックコンポーネント:

class TestComponent < HyperComponent
  render(DIV) do
    'Its works'
  end
end
