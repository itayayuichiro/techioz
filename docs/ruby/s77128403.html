Ruby on Rails のデフォルトの情報ページを維持するにはどうすればよいですか?

概要

Ruby on Rails の「デフォルトの情報ページ」を維持する方法を探しています。 Rails 7 では、Rails ロゴが中央に表示され、Ruby や Rails などのバージョン番号が下部に表示されるページです。config/routes.rb ファイルにルートを追加すると、デフォルトのページは表示されなくなります。 。自宅での簡単な実験プロジェクトのインデックスページにしたいと思っています。何か案は？おそらくそれは Puma Web サーバーに関連付けられていますか?ありがとう！

解決策

このページは Rails からのもので、WelcomeController によって処理されます。これを維持するには、次のルートをルート ファイルに追加するだけです。

  get 'your-unique-path', to: 'rails/welcome#index'

  # or

  root to: 'rails/welcome#index'
