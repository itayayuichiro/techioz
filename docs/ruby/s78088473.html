ページにルビって？

概要

私のマシンでは、Ruby の ri データは /usr/share/ri/3.0.0/system にあります。

その中には次のようなものがあります。

ri Array や ri Array#clear などのコマンドを使用して、クラスまたはメソッドの情報を取得できます。

ファイル page-dig_methods_rdoc.ri から情報を取得するにはどうすればよいですか?

解決策

当然です。友人が私にこう言いました：ri Ruby:dig_methods。
