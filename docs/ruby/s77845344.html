Ruby バージョンマネージャー

概要

私は Ruby を初めて使用しており、現在 Windows 10 を使用しています。RVM のインストールで困難に直面しています。私は Ruby のバージョンが異なるさまざまなプロジェクトに取り組んでいるため、RVM を使用してそれらを簡単に管理したいと考えています。現在、プロジェクトを切り替えるたびにシステム変数のパスを頻繁に変更していることに気づきました。これについて手伝ってもらえますか？前もって感謝します！

RVMをインストールしましたが、次のようなものが表示されます 「rvm」は内部コマンドまたは外部コマンドとして認識されません。 動作可能なプログラムまたはバッチファイル。

解決策

RVM は、ほとんどの UNIX などのシステムと Windows をサポートしています。基本的な要件は、bash、curl、gpg2、およびツールの GNU バージョン全体ですが、RVM はそれを自動検出し、必要なものをすべてインストールしようとします。

WSL (Linux 用 Windows サブシステム) を使用する - https://learn.microsoft.com/pt-br/windows/wsl/install

から: https://rvm.io/rvm/install

gpg --keyserver hkp://keys.gnupg.net --recv-keys 409B6B1796C275462A1703113804BB82D39DC0E3 7D2BAF1CF37B13E2069D6956105BD0E739499BDB

sudo apt-get update && sudo apt-get install -y gnupg2

gpg --keyserver keyserver.ubuntu.com --recv-keys 409B6B1796C275462A1703113804BB82D39DC0E3 7D2BAF1CF37B13E2069D6956105BD0E739499BDB

\curl -sSL https://get.rvm.io | bash -s stable --ruby

rvm install <ruby version>

rvm list

rvm use <ruby version>
