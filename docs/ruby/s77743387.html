Ransack::Search のインスタンスの未定義メソッド エラー

概要

新しい Rails アプリを作成しています。active_admin gem を使用してユーザー用の管理ページを作成しました。ページにアクセスしようとすると、このエラーが発生し続けます

ActionView::Template::Error (undefined method `email_equals' for an instance of Ransack::Search):
    1: # frozen_string_literal: true
    2: insert_tag renderer_for(:index)

ransack (4.1.1) lib/ransack/search.rb:114:in `method_missing'

ここがユーザーの管理ページです

ActiveAdmin.register User do
  index do
    id_column
    column :current_sign_in_at
    column :sign_in_count
    column :created_at
  end
end

これは、以前にエラー参照を修正した後に発生しました: Ransack には、 関連付けが検索可能として明示的に許可リストに登録されている必要があります

解決策

Ransack バージョン 4 では、属性と関連付けの許可リストへの明示的な記載が必須となっているため、これらの問題が発生します。

https://translate.google.com/translate?hl=ja&sl=en&tl=ja&u=https://github.com/activerecord-hackery/ransack/releases/tag/v4.0.0

これを修正するには2つの方法があります

    def self.ransackable_associations(_auth_object = nil) = %w[events]
    def self.ransackable_attributes(_auth_object = nil) = %w[state]

    config.filters = false
