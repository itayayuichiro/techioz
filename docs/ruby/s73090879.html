macOS で Ruby のデフォルトを 3.1.2 にする

概要

私はmacOSで作業しています。

brew upgrade ruby
Warning: ruby 3.1.2 already installed

でもやってる

ruby -v

私を捕まえるだろう

ruby 2.6.3p62 (2019-04-16 revision 67580) [universal.arm64e-darwin20]

私は試した

whereis ruby

そしてそれは私に与えてくれています

/usr/bin/ruby

Ruby -v を実行するとバージョン 3.1.2 が表示されるようにしたいのですが、マシンのデフォルトをバージョン 3.1.2 にする方法がわかりませんでした。なにか提案を？

解決策

rbenv や RVM などの Ruby バージョンマネージャーを使用すると簡単になるかもしれません

brew install rbenv 
rbenv install 3.1.2
rbenv use 3.1.2

それ以外の場合は、この回答を確認できます: https://stackoverflow.com/a/8731098/14473816

編集：

ラデベ王子はこう言った。

rbenv use 3.1.2 の代わりに rbenv local 3.1.2 を使用する必要があります。
