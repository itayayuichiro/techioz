概要

現在、次のものを使用してプロジェクトに取り組んでいます。

解決策

バージョン 10.11.x 以降の Mac OSX で作業している場合は、次のアプローチがうまくいきました。

gem install nio4r -v '1.1.0' -- with-cflags="-std=c99"
