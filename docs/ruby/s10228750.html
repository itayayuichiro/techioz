Rails 7 アプリでの ActionText のレンダリングの問題

概要

<%= form_for @user1 do |f| %>

    Username: <%= f.text_field :username %>      <br />
    Password: <%= f.password_field :password %>  <br />.
    Email:    <%= f.email_field :email %> <br />

    <%= submit_tag "Add" %>

<% end %>

私は Ruby の初心者で、簡単なプログラムを作成しようとしていますが、このエラーが発生し、その理由がわかりません。このエラーが発生しました

undefined method `users_path' for #<#<Class:0x000000020576f0>:0x0000000335d5d8>

ここでの解決策は何ですか？

class UserController < ApplicationController
  def add
    @user1 = User.new
    respond_to do |format|
      format.html
      format.json {render :json => @user1}
    end
  end

解決策

config/routes.rb でルートを定義しましたか?

Rails.application.routes.draw do
  resources :users
  # ...
end
