解決策

さて、わかりました。実際、Rails 7.0.4 までは、Redis >= 3、< 5 が Action Cable Redis サブスクリプション アダプターの要件です。

ファイルの先頭にある https://github.com/rails/rails/blob/v6.1.7/actioncable/lib/action_cable/subscription_adapter/redis.rb を参照してください。

なぜこの問題に遭遇する人がこれほど少ないのか不思議です。とにかく、解決策は、Rails >= 7.0.4 (これはかなり大幅です) にアップグレードするか、redis gem を < 5 にダウングレードすることです。

後者はすでに実行したとおっしゃっていましたが、何か問題があったようです。
