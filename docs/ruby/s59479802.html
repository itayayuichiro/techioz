解決策

私も同じ問題を抱えていました。以下を使用して msgpack gem を完全に削除する必要があります。

gem uninstall msgpack

次に、それを再インストールして、現在使用している Ruby のバージョンに対してライブラリがビルドされるようにします。

次のことも試してみてください:

bundle install --redownload

これにより、gem が再インストールされ、現在使用している Ruby のバージョンでリンクされたライブラリが再構築されます。
