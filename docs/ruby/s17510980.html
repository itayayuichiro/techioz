Ruby スプラットと << 演算子

概要

これをやりたいです:

a << *b

しかし、これはirbで起こります:

1.9.3p327 :020 > a
 => [1, 2, 3, 4] 
1.9.3p327 :021 > b
 => [5, 6, 7] 
1.9.3p327 :022 > a << *b
SyntaxError: (irb):22: syntax error, unexpected tSTAR
a << *b
      ^

何かが足りないのでしょうか？

解決策

この行はあなたの仕事を行います:

a += b
