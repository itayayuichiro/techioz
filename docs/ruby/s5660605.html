RVM を無効にするか、RVM なしでインストールされた Ruby を使用しますか?

概要

Rails アプリケーションには Ruby 1.8.7 と Rails 2.3.11 をインストールしました。そしてそれはうまくいきました。その後、RVMとREEをインストールしました。 したがって、Ruby スクリプト/サーバーの実行中は REE を使用します。しかし今は、RVM なしでインストールされた古い Ruby 1.8.7 を使用したいと考えています。

あらゆる解決策！

解決策

Ruby 経由で管理されていないシステムを使用するには、rvm use system を試してください。 rvm。
