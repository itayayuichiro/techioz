RubyMine は Ruby がインストールされていないと言っていますが、実際にはインストールされています

概要

RubyMine で Rails プロジェクトを開くと、「必要な Ruby-3.3.0 がインストールされていません」というエラー メッセージが表示されます。ただし、コマンド Ruby –version を入力すると、ターミナルには Ruby 3.3.0 が出力されます。どうしたの？

解決策

RubyMine では、各プロジェクトの設定で SDK を設定する必要があります。

タイトルバーの「Rubymine」をクリックします。

「言語とフレームワーク」の開閉用三角ボタンをクリックします。

「Ruby SDK と Gems」をクリックします。

インストールされている Ruby のリストが表示されるはずです。 Ruby のラジオ ボタンを選択し、[OK] ボタンを押します。
