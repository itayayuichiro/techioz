Bundler を使用して新しいバージョンの Bundler をインストールする

概要

Bundler could not find compatible versions for gem "bundler":
  In Gemfile:
    bundler (>= 1.10.2) ruby

  Current Bundler version:
    bundler (1.9.9)
This Gemfile requires a different version of Bundler.
Perhaps you need to update Bundler by running `gem install bundler`?
Could not find gem 'bundler (>= 1.10.2) ruby in any of the sources

gem install Bundler と入力するのではなく、bundler に新しいバージョンの Bundler をインストールするように依頼することはできますか?

解決策

バンドラーはそれ自体をバンドルできないようです:)

したがって、sudo gem install Bundler を実行する必要があります。
