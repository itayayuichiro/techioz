gnupg2 が見つからず、Mac では再インストールが役に立たなかった

概要

Mac 用の RVM をインストールしようとしていますが、すでに gnupg/gnupg2 がインストールされています。

 brew install gnupg2

そして私はコマンドを使用しました

 gpg2 --recv-keys 409B6B1796C275462A1703113804BB82D39DC0E3 7D2BAF1CF37B13E2069D6956105BD0E739499BDB

誰か答えてくれませんか？ありがとう。

解決策

gnupg2 に加えて gnupg をインストールする必要がある場合があります

https://stackoverflow.com/a/27042267/4701098を参照してください。
