Rails ルーティング: root に (浅い) 懸念を追加

概要

私の Routes.rb では、以下のような懸念を定義しました。

concern :namespaceable do
  resources :comments do
    resources :replies
  ...

これで、この懸念を任意のリソースに追加できるようになりました。

resources :groups, concerns: :namespaceable, shallow: true

しかし、この懸念をルートに浅く追加して、次のルートを作成できるようにするにはどうすればよいですか

どうもありがとう！

解決策

メインファイルに懸念事項を書き込むだけです::namespaceable

Rails.application.routes.draw do
  concern :namespaceable do
    resources :comments do
    resources :replies
    ...
  end

  concerns :namespaceable # Will add to root namespace
end
