複数のスキーム環境用のポッドのインストール

概要

私は iOS プログラミングの初心者で、現在、現在のアプリ用に複数の環境を構築したいと考えています。しかし、ポッドのインストール時には成功しますが、ポッドが認識されません。エラーが表示される

私がインストールするすべてのフレームワークで。

したがって、私のスキームは次のようになります。

そして、これが私の構成ファイルの設定です:

そしてこれが私のPodfileです:

post_install do |installer|
    installer.pods_project.build_configurations.each do |config|
        config.build_settings.delete('CODE_SIGNING_ALLOWED')
        config.build_settings.delete('CODE_SIGNING_REQUIRED')
        config.build_settings['SWIFT_VERSION'] = '4.0'
    end

    installer.pods_project.targets.each do |target|
        if ['Charts','DropDown', 'RealmSwift', 'Pulley', 'LTHRadioButton'].include? "#{target}"
            print "Setting #{target}'s SWIFT_VERSION to 3.0\n"
            target.build_configurations.each do |config|
                config.build_settings['SWIFT_VERSION'] = '4.2'
            end
        else
            print "Setting #{target}'s SWIFT_VERSION to Undefined (Xcode will automatically resolve)\n"
            target.build_configurations.each do |config|
                config.build_settings.delete('SWIFT_VERSION')
            end

どうすればいいですか？ありがとう。

解決策

ポッド ファイルに移動してポッドを定義し、特定のターゲットに追加します。

def pods
    pod 'Pod 1', '~> 1.0'
end

target 'Target 1' do
    pods
end

target 'Target 2' do
    pods
end
