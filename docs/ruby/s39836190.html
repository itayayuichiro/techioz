Homebrew のインストール: git fetchorigin master:refs/remotes/origin/master -n – Depth=1 中に失敗しました

概要

ターミナル（Mac El Capitan）で次のコマンドを実行しました

$ /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

次のエラーメッセージが表示されました。

Failed during: git fetch origin master:refs/remotes/origin/master -n --depth=1

さらに試してみた:

sudo chown $(whoami):admin /usr/local && sudo chown -R $(whoami):admin /usr/local

次に、上記の 2 つのコマンドを実行します。

Homebrewをインストールする方法はありますか?

brew が認識されず、ruby バージョン 2.0.0、git バージョン 2.7.4

解決策

数時間の調査と総当たりの結果、問題の原因は git が適切に設定されていないことであることがわかりました。一部の記事では、git をダウングレードするか、すべてを再インストールすることを提案しています。ただし、次を追加するだけで問題を解決しました

git config --global user.email [email protected]

次に、cellar ディレクトリと homebrew ディレクトリを /usr/local から削除し、インストールを再試行します。
