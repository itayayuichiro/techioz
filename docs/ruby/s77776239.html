-bash: rbenv: コマンドが見つかりません

概要

Ruby rbnevを介してjekyllをインストールしようとしていますが、open ~/.bash_profileをターミナルに置き、テキストエディタにeval ““を入力した後、これがその中にある唯一のもので、すべてを閉じてターミナルを開くと、上記のエラーメッセージが表示されます-bash: rbenv: コマンドが見つかりません。 rbenv コマンドは機能せず、gem install jekyll を実行すると次の結果が得られます。

ERROR:  While executing gem ... (Gem::FilePermissionError)
    You don't have write permissions for the /Library/Ruby/Gems/2.6.0 directory.

インターネット中を探しましたが、助けが見つかりませんでした。助けていただければ幸いです。

ほぼすべてを試しました。私はこれで20分間立ち往生しています。 「問題の詳細」で自分のパスを詳しく説明しました

解決策

タイプミスをしました。 .bash_profile の行は次のようになります。

eval "$(rbenv init - bash)"
                    ^-- NOTE THE SPACE HERE
