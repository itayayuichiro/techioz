解決策

私の醸造は次のように設定されました：

which brew
/usr/local/bin/brew

これが原因で x86 でインストールされていました。

次のように設定されていることを確認しました。

which brew
/opt/homebrew/bin/brew

M1 Mac に arm64 を使用して適切にインストールするには。

これにより、Ruby を正常にインストールするための openssl、libyaml、psych などの問題が解決されました。
