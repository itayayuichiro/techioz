「不明な Ruby インタプリタのバージョン (対処方法がわかりません): RUBY_VERSION」の修正方法。

概要

今日、Mac に jekyll ブログをセットアップしたいと思っています。すでに Ruby 2.3.0 をインストールしていますが、「$ jekyllserve」を作成するとエラーになります。これをターミナルに表示します:

エラー情報:

Unknown ruby interpreter version (do not know how to handle): RUBY_VERSION.

解決策

だからそれは次のようになります

bundle exec jekyll new

次の行で Gemfile を作成します

ruby RUBY_VERSION

そのファイルを次のように編集するとよいでしょう。

ruby '2.1.1'
