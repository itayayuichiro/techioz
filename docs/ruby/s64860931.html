rbenv が最新の Ruby バージョンを Ubuntu にインストールできないのはなぜですか?

概要

これらの新しいバージョンが Ubuntu で表示されないのはなぜですか? Ubuntu に rbenv 1.1.1 がインストールされています。

これはおそらく愚かな質問ですが、Webdev レベル以下のコードについてはほとんど理解していません。

解決策

このコマンドを実行すると問題が解決しました。

git clone https://github.com/rbenv/ruby-build.git ~/.rbenv/plugins/ruby-build

Ruby の最新バージョンが表示されます。
