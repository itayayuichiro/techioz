インテリセンスのような提案がどこから来たのかを判断するにはどうすればよいでしょうか? (そしてそれを抑制します) [重複]

概要

Ruby コードを作成するとき、カーソルが end ステートメントに到達するたびに、次の提案が表示されます (インテリセンスに似ています)。

これは非常に迷惑であり（エスケープする必要があるため、流れが中断されます）、役に立ちません。それがどこから来たのか、どの拡張子から来たのかわかりません。

この提案を挿入するプラグインの部分を特定するにはどうすればよいですか?

それで、どうやってそれを遮断すればいいのでしょうか？

解決策

settings.json ファイルでは、どのプラグインが推奨事項を作成するかを制御できます。私の場合は次のとおりです。

"editor.quickSuggestions": null // use rubocop for formatting
"ruby.intellisense": "rubyLocate",

インテリセンス機能を純粋に無効にする場合は、これらの設定によって制御されます。

https://translate.google.com/translate?hl=ja&sl=en&tl=ja&u=https://code.visualstudio.com/docs/editor/intellisense

私は、純粋に「エンド」インテリセンスのみをオフにできるとは信じていません。これは、全か無かの命題のようです。
