ターミナルでこの一見基本的なステップ「Homebrew を PATH に追加する」を実行するにはどうすればよいですか?

概要

私はこの種の仕事にはまったく慣れていません。 M1 MacBook に Homebrew をインストールしようとしているときに、誰かがターミナルでこのメッセージを簡単に段階的に翻訳してくれることを願っています。

…

…

解決策

ターミナルのメッセージに示されている 2 つのコマンドを実行するだけです。こうすることで、homebrew は現在実行中のシェルだけでなく、開いた新しいシェルに対しても正しくセットアップされます。
