プロキシの背後で iruby をインストールすると、必要な msys2 パッケージのインストールに失敗する

概要

オフィスのコンピューターに iruby をインストールしようとしましたが、「必要な msys2 パッケージのインストール: mingw-w64-ucrt-x86_64-zeromq」で失敗し、次のエラー メッセージが表示されて必要なファイルをロードできませんでした: ミラーへの接続に失敗しました。 ustc.edu.cn ポート 443 10016 ミリ秒後: タイムアウトに達しました

プロキシをどこに設定すればよいですか?また、このエラーを回避するにはどうすればよいですか?

すでに MSYS2 をインストールし、適切なプロキシを設定し、MSYS2 を更新し、その中に mingw-w64-ucrt-x86_64-zeromq をインストールしました。 iruby をインストールしようとすると、管理モードで PowerShell 7.4 を使用し、次のステートメントを入力します。

gem install iruby --http-proxy proxy.mycompany.de:port

次に、これらの行を取得して待ちます。
