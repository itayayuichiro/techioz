Mac OS X 10.9.5 に Ruby 2.1.3 をインストールする際のワーキング ツリー エラーを検査またはクリーンアップする

概要

rbenv install 2.1.3 コマンドを使用して、Mac OS X 10.9.5 に Ruby 2.1.3 をインストールしようとしています。ただし、以下のエラーメッセージが表示されます。スタックオーバーフローなどに関するあらゆる提案を試しました。何も機能していないようです。現在、10.9.5 に付属の元の Ruby バージョンである Ruby 2.1.3p242 を使用しています。 Brew Doctor は、すべて問題なく、rbenv は最新であると言っています。本当にありがとう！ Ruby と Ruby on Rails を学ぼうとしていますが、この段階をクリアできません。

解決策

私は m1 Mac を持っていますが、Ruby のインストール中にこのエラーが発生しました。 このコマンドは私にとってはうまくいきました：

export optflags="-Wno-error=implicit-function-declaration";

これを実行してから、rbenv インストールを再試行してください。
