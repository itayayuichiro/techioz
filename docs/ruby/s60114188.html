rubocop-rspec から少数の警官だけを使用するにはどうすればよいですか?

概要

宝石rubocop-rspecからいくつかの警官だけを使用したいと思います。

この方法を試してみましたが、うまくいきません:

require:
  - rubocop-rspec

RSpec:
  Enabled: false

RSpec/ExampleWithoutDescription:
  Enabled: true

私もこれを試してみました:

require:
  - rubocop-rspec

AllCops:
  RSpec:
    Enabled: false

RSpec/ExampleWithoutDescription:
  Enabled: true

解決策

ドキュメントに従って、デフォルトですべての警官を無効にすることができます。

AllCops:
  DisabledByDefault: true
