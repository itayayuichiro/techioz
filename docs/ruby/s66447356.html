Ruby 3.0 の IRB でデフォルトの構文の色付けを変更するにはどうすればよいですか

概要

最新の Ruby 3.0 を使用できることにとても満足しています。さらに、構文の強調表示と色分けを行う更新されたコマンドライン インタープリターにアクセスできます。

ただ、私には色が少し見にくいです。どうすれば変更できますか? IRB のコマンド ライン オプションを使用すると、–nocolorize を使用して色付けをオフにできますが、デフォルトを変更できる (たとえば、青色を明るくするなど) 設定ファイルがどこにあるのかわかりません。

解決策

iTerm2のテーマを変更することでこれを修正しました。

「Tango Dark」に切り替えると読みやすくなりました。
