JavaScript 配列: 項目の「範囲」を取得します

概要

JavaScript で Ruby の array[n..m] に相当するものはありますか?

例えば：

>> a = ['a','b','c','d','e','f','g']
>> a[0..2]
=> ['a','b','c']

解決策

array.slice(begin [, end]) 関数を使用します。

var a = ['a','b','c','d','e','f','g'];
var sliced = a.slice(0, 3); //will contain ['a', 'b', 'c']

最後のインデックスは包括的ではありません。 Ruby の動作を模倣するには、終了値をインクリメントする必要があります。したがって、slice は Ruby の a[m…n] のように動作すると思います。
