英国の鉄道情報 API? [閉まっている]

概要

計画中の英国の観光イベント アプリに鉄道旅行の料金詳細を追加したいと考えています。

電車の時刻表と料金の詳細を提供する API を知っている人はいますか? thetrainline.com や Raileasy を通じて得られる情報のようなものですが、予約機能は必要ありません。 Googleはこの件に関してあまり情報を出していません…

必要に応じてラップできるので、API が何に書かれているかについてはあまり気にしません:)

解決策

National Rail Enquiries を運営する ATOC がこのデータを所有しています。彼らからライセンスを取得する必要があります。一般的な利用規約はここにあります。

それが非営利の取り組みである場合、http://traintimes.org.uk/ で行っているように、スクリーンスクレイピング型の操作には目をつぶるかもしれません。
