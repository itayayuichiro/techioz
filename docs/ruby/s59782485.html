sudo gem install Cocoapods を試してみましたが、助けがありませんでした。

同様の問題に直面し、解決策を見つけた人はいますか?

解決策

gem xcodeproj をインストールし、ターミナルで実行する必要があるようです。

gem install xcodeproj
