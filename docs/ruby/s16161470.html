Rails 3.0 でのユーザー通知メールに mailchimp を使用できますか?

概要

パスワードリセットメール、アカウント確認メール、ユーザーが別のユーザーや他のすべてのシステムからメッセージを受信したときのようなその他のメールなど、rails 3.0 のシステムに mailchimp を使用できますか?

解決策

あなたが言及している電子メールの種類は「トランザクション電子メール」と呼ばれ、MailChimp は「バルク電子メール」用です。いいえ、そのために MailChimp を使用することはできません。それは実際には利用規約に反しています。

しかし、アトランタの優秀な人々が Mandrill を立ち上げたのはつい昨年のことです。マンドリルは、ディエゴ マラドーナがサッカー ボールを扱うように、トランザクション電子メールを扱います。

Mandrill は、素晴らしいドキュメントを備えた多数の API メソッドを提供します。
