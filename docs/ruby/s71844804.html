この後、pod install または pod install –repo-update が問題の解決に役立つことを期待して、iOS サブフォルダーに移動します。しかし、どちらのインスタンスでも、次の行の後にすべてのエラー行が表示され続けます。

 /Library/Ruby/Gems/2.6.0/gems/cocoapods-1.11.3/lib/cocoapods/user_interface/error_report.rb:34:in `force_encoding': can't modify frozen String
(FrozenError)

これらすべてのエラーは、XCode で SDK 13.0 以降を使用するために、Monterrey にアップグレードする必要があった後に発生しました。

ちょっとしたメモですが、このアップグレードの前にいくつかのビルドを作成しましたが、それらは問題なく Apple Store にアップロードされました。

解決策

私にとってはうまくいきました。ターミナルでこのコードを実行してください。

sudo gem update xcodeproj
