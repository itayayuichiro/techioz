ランダムな単語から作られた Heroku アプリの URL

概要

私は Rails と Heroku の初心者で、これはばかげた質問のように思われるかもしれません。そうであれば申し訳ありませんが、少なくとも誰かが私にそう言ってくれれば幸いです。

Heroku の「はじめに」ガイドに従い、Rails のデフォルト プロジェクトのコピーをデプロイすることができました。しかし、heroku によって事前に提供された URL は http://radiant-ice-84.heroku.com でした。私の質問は次のとおりです。

heroku はランダムに生成された URL を割り当てますか、それとも radiant-ice-84 を認識しないために何か間違ったことをしたのでしょうか?

解決策

この質問を閉じる前に、実際には、アプリを作成するときに次を使用して最初の値を指定できることをお伝えしたいと思います（設定ページで再度変更する必要はありません）。

heroku create your-app-name
