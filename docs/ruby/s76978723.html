Rails コンソールが連続する改行を削除しないようにする

概要

Rails コンソールが連続した改行を削除しないようにしたいと考えています。

CoolApp (PRODUCTION) > a = '
CoolApp (PRODUCTION) " hi
CoolApp (PRODUCTION) " 
CoolApp (PRODUCTION) " 
CoolApp (PRODUCTION) " hi'
=> "\nhi\nhi"

誰か私がそれを行う方法を知っていますか？上の文字列は「」である必要があります。 こんにちは

こんにちは”

解決策

新しい行ごとに単一のスペースを入力すると、その行の改行が保持されるようです:)
