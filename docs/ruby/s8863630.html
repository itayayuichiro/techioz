nanoc のようなシステムに代わる Python は何でしょうか?

概要

nanoc に似た Python パブリッシング システム (これがそのようなものに適切な名前かどうかわかりませんが、彼らはそう呼んでいます) はありますか?一般的に、大量のマークアップ/asciidoc ファイルを順序立てて HTML に変換するものでしょうか?

Python-markdown については知っていますが、外部の .css ページをサポートしない 1 つずつのページは私が探しているものではありません。

では、多かれ少なかれこの品質に基づいた Python ベースのものはあるのでしょうか?

解決策

一部の Web サイトは、ドキュメント化を目的とした Sphinx を使用して構築されていますが、静的な Web サイトに簡単に拡張できます。 (例: http://redhotchilipython.com/)
