ASDF に Ruby のバージョンがありません

概要

asdf 経由で Ruby 2.7.7 をインストールしようとしていますが、最新バージョンは 2.7.6 のようです。 asdf で新しい Ruby バージョンをサポートする方法を知っている人はいますか?

利用可能な Ruby 2.7.x バージョン

Ruby 2.7.7 をインストールしてみました。利用できると予想されるバージョンです。

解決策

asdf-ruby プラグインは、ruby-build を使用して実際に Ruby バージョンをビルドします。新しい Ruby バージョンをインストールできるようにするには、ruby-build を更新する必要があります。

asdf を呼び出す前に ASDF_RUBY_BUILD_VERSION 環境変数を目的のバージョンに設定することで、目的の Ruby ビルド バージョンを設定できます。

または、asdf-vm/asdf-ruby#317 でデフォルトで現在の Ruby-build バージョンを使用するようにプラグインが更新されているため、すべての呼び出しで現在の Ruby-build バージョンを使用するように asdf-ruby プラグインを更新することもできます。

これはどちらでも実行できます

asdf plugin update --all

asdf plugin update ruby

Ruby プラグインを更新するだけです。
