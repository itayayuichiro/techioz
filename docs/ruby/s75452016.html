Ruby でインストールが失敗しました: jekyll をインストールできません

概要

Ruby に依存する Jekyll を使用して GitHub ページをセットアップしようとしていました (ここのように)。残念ながら、インストールでちょっと失敗したようです（数回 sudo apt-get install Ruby-full と sudo apt-get Remove Ruby-full を実行した後…）

現在、コンピューター上に古いバージョンの Ruby (と思われる) が残っており、更新できません。現在の私のシステムの状態は次のとおりです。

多くの削除/インストールを試し、更新された gem や Ruby を試しました。これまでのところ何もロックを解除できませんでした。

解決策

Ruby3.0 を置き換えてみてください。sass1.58 は高度すぎて、ruby2.7 と互換性がない可能性があります。 次の手順に従ってコマンドを実行してください。
