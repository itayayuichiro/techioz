PayPal の支払い、サンドボックスで「送信者の資金が不足しています」

概要

PayPal 支払いを設定しようとしています。 API 呼び出しは完了していますが、サンドボックスでは次のメッセージが表示され続けます: 送信者の資金が不足しています。申請に資金を追加するにはどうすればよいですか?送信元のアカウントを指定するオプションが見つからないようです。

解決策

おそらく答えは、複数の通貨 (たとえば、ユーロ) を持っていて、ユーロ残高が 1000 あるのに、米ドルで支払おうとしているということです。 したがって、テストアカウントを使用してsandbox.paypal.comにログインし、通貨を平準化して、ユーロを米ドルに送金し、支払いを再試行すると、成功するはずです。
