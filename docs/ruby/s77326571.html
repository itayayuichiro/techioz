Ruby の Redis クライアントでブルームフィルターを使用する方法

概要

存在するすべての Ruby ライブラリはカスタム ブルームフィルター実装を行っています。 Redis のネイティブ ブルームフィルター機能を使用したいと考えています。

Ruby クライアントを使用して手動コマンドを Redis に送信する方法はありますか?

解決策

これらのメソッドを手動で呼び出すことで使用できます。例えば：

client.call("BF.ADD", bloom_filter_name, key)

client.call("BF.EXISTS", bloom_filter_name, key)

上記の注意事項を誤解しないでください。
