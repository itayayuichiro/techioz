rbenv を使用する OS X Mavericks 上の mkmf.log ファイルはどこにありますか

概要

私は OS X Mavericks を実行しており、Ruby のバージョン管理に rbenv を使用しています。たとえば、Ruby バージョン 1.9.3-p448 の場合、ファイル mkmf.log はどこにあるのでしょうか?

解決策

rmagick バージョン 2.11.1 と Ruby バージョン 1.9.3-p448 の場合、ファイルはここにありました。

~/.rbenv/versions/1.9.3-p448/lib/ruby/gems/1.9.1/gems/rmagick-2.11.1/ext/Rmagick

そのパスの最後の部分 /Rmagick は、すべての gem に適用されるわけではありません。 mkmf.log ファイルはパスの最後にある /ext のすぐ内側にあると思います。
