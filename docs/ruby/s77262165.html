Rails 7 Flatpicr の無効化時刻と日付

概要

現在、予約アプリの開発に取り組んでおり、すでに作成された予約に基づいて時間を無効にしようとしています。ただし、丸 1 日または特定の日の範囲を無効にするドキュメントのみが表示されます。このようなことは可能でしょうか？そうでない場合、Rails 7 でこれを処理できるライブラリはありますか?ありがとう！

flatpickr_controller.js

現在サポートされているもの:

 disable: [
        {
            from: "2023-10-10",
            to: "2023-10-10"
        },
       
    ]

私の要件:

 disable: [
        {
            from: "2023-10-10 1:00 PM",
            to: "2023-10-10 2:00 PM"
        },
       
    ]

解決策

まだその機能はないと思うよ、CMIIW
