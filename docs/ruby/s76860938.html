Ruby 3.2 アップグレード、therubyracer gem はバンドルできない

概要

現在、アプリケーションの Ruby バージョンを 3.2 にアップグレードしていますが、therubyracer をバンドルできないという問題が発生しています。

この gem は Ruby 3.2 と互換性がありますか?

ログに記録される内容はそれほど多くありません。
