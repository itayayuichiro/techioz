EL Capitan 10.11.1 にココア ポッドをインストールできない

概要

次のエラーが発生します:

エラー: 有効な gem ‘cocoapods’ (>= 0) が見つかりませんでした。理由は次のとおりです。 https://rubygems.org/ からデータをダウンロードできません - Errno::ETIMEDOUT: 操作がタイムアウトしました - connect(2) (https://rubygems.org/latest_specs.4.8.gz)

助けてください。

解決策

まず、システムを更新する必要があります (これにより、Ruby gem がインストールされます)。

アップデートシステム

sudo gem update --system

COCOAPODS をインストールする

ELキャピタン以前

sudo gem install cocoapods

ELキャピタン後

sudo gem install -n /usr/local/bin cocoapods

セットアップポッド

pod setup

POD INIT (プロジェクトディレクトリに移動)

pod init

ポッドファイルを開いて更新する

open -a Xcode Podfile

必要に応じてポッドをインストールします

pod install

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

export PATH=/opt/homebrew/bin:$PATH

brew install ruby

brew install cocoapods

pod --version

pod install
