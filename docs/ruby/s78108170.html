WSL 2 Ubuntu はそのようなファイルをロードできません – zlib

概要

古いプロジェクトを入手しました。しかし、ガイドから Bundle install コマンドを実行すると、このエラーが発生しました。

irwan@LAPTOP-3KM80MDK:~$ gem install bundler:1.9.0 
ERROR:  Loading command: install (LoadError)
cannot load such file -- zlib ERROR:  While executing gem ... (NoMethodError)
undefined method `invoke_with_build_args' for nil:NilClass

もう走ってます

sudo apt install zlib1g-dev

しかし、私はまだ同じエラーを抱えています。インストールパスが間違っているような気がするのですが、確認方法がわかりません。

解決策

これを行うことで修正しました:

sudo apt install zlib1g-dev

その後

rbenv uninstall 2.3.3
rbenv install 2.3.3

2.3.3を再インストールする必要があります
