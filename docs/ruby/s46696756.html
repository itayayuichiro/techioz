外部ライブラリ (gem) を Rubymine のプロジェクト ビュー内に表示するにはどうすればよいですか?

概要

Rails アプリ内でできるのと同じように、Cmd キーを押しながらメソッド名をクリックして gem のソースコードにジャンプできるようにしたいと考えています。

過去に Rubymine を使用したことがありますが、左側の「プロジェクト ビュー」の下部にある「外部ライブラリ」というタイトルのすべての gem のフォルダーが自動的に読み込まれたことを覚えています。何らかの理由で、下の写真にあるものしか取得できません。このプロジェクトの .ruby バージョンが 2.3.1 であるのに 2.0.0 しか表示されないため、ある種の RVM の問題のような気がします。

解決策

[ファイル] -> [設定] -> [言語とフレームワーク] -> [Ruby SDK と Gems] に移動します。次に、コードで使用している Ruby の正しいバージョン (この場合は 2.3.1) を選択します。これにより、プロジェクト ウィンドウの「外部ライブラリ」セクションが正しいバージョンの gem ホームからロードされるようになります。必要に応じて Rubymine を再起動します。
