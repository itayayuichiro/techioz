m1 mac に jekyll (eventmachine) をインストールできない

概要

私は iOS 開発の初心者で、新しい GitHub ブログを作成したいと考えています。

参考としてApple M1 Macbookにjekyllをインストールする方法に従っています。

そして、M1 Pro Macbook (macOS Monterey 12.6) に Jekyll をインストールしようとしましたが、 gem install –user-install Bundler jekyll を使用して Bundler と Jekyll をインストールすると失敗します (以下の出力)。

私はすでに、最終的に成功した多くのブロガーの記事を読みました。この問題はどうすれば解決できますか?誰か助けてください！
