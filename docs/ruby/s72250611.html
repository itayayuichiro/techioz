apt-get を使用して openssl バージョン 1.1 をインストールしようとしましたが、機能しません。

解決策

Ruby のコンパイル中に openssl の問題が発生した場合、私は通常、このような解決策を使用します。私はできることなら、インストールされている OS のバージョンをいじりたくないので、RVM などは使用しません。

wget https://www.openssl.org/source/openssl-1.1.1q.tar.gz

tar xvzf openssl-1.1.1q.tar.gz
cd openssl-1.1.1q
mkdir -p $HOME/lib/openssl
./config --prefix=$HOME/lib/openssl
make
make install

cp /etc/ssl/certs/* $HOME/lib/openssl/ssl/certs

tar xvzf ruby-2.7.1.tar.gz
cd ruby-2.7.1
./configure --prefix=$HOME/.rubies/ruby-2.7.1 --with-openssl-dir=$HOME/lib/openssl
make
make install
