VSCode での Heredoc 構文の強調表示の追加

概要

View Components を使用している Rails プロジェクトがあります。ほとんどのコンポーネント内には、ERB テンプレート化された HTML を使用してコンポーネントのマークアップを定義する erb_template HEREDOC があります。

HEREDOC 開始タグを HTML (大文字のみ) に設定すると、HTML 構文が強調表示されます。同様に、RUBY に設定すると、ルビの構文が強調表示されます。

この種の埋め込み構文の強調表示をどこで設定し、ERB サポートを追加できるかを見つけるのに苦労しています。これを成功させた人はいますか?

解決策

ヒアドキュメントへの言語の追加については、https://github.com/Shopify/vscode-ruby-lsp/issues/820 を参照してください。

grammars/ruby.cson.json を変更して、vscode-ruby-lsp に PR を送信できると思います。ファイルは以前は https://github.com/atom/ language-ruby/ にありましたが、リポジトリはロック/アーカイブされているように見えます。
