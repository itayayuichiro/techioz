Mac に React Native をインストールできない

概要

gem install bundler

解決策

Ruby アップデート修正のリンクについては @engineersmnky に感謝します。特に次の行を使用しました。

rvm install ruby-3.3.0 --reconfigure --with-openssl-dir=$(brew --prefix openssl@3)

これにより、実際に Ruby を更新できるようになりましたが、これが本当の問題でした。その後、プロジェクトとiosにcdして実行できました

bundle install
bundle exec pod install

そしてそれは正しく実行されました
