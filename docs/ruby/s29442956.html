「requirements_debian_update_system Ruby-1.9.3-p551」の実行中にエラーが発生しました。

概要

RVM install 1.9.3 のインストール時に直面する問題 ubuntu 12.04で

私を助けてください

解決策

ここでの問題は、ubuntu RVM の手順に従って再起動 (またはログアウト) しなかったため、ユーザーが (まだ) RVM グループの一部ではないため、ログ ディレクトリへの書き込み権限がないことです。 この問題は、ログアウト、再起動するか、次のコマンドを使用して新しいシェルを開くことで修正できます。

sudo su - $USER

これにより、正しい権限を持つユーザーで新しいシェルが作成されます。
