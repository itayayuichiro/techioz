Wkhtmltopdf 単一行の文字がページ間で部分的に切れている

概要

私は、Ruby on Rails(3.1) を使用するプロジェクトで作業しています。私の要件は、HTML コンテンツから PDF を生成することです。そこで私はpdfkit gemを使用します。

一部のページでは、単行の文字がページ間で部分的に切れています。 htmlを変換すると、pdfkit gemを使用してpdfに変換します

wkhtmltopdf のバージョン: wkhtmltopdf – 0.11.0 rc1

オペレーティング システム: Linux CentOS 5.5

下の画像では、ページ間で文字が部分的にカットされています。

解決策を提案してください。

解決策

テーブルに関してこの問題が発生しました。

次に、これを CSS に追加しました。

table, img, blockquote {page-break-inside: avoid;}

これにより問題が解決されました。
