Ruby のバージョンが両方の React Native プロジェクトで異なります

概要

システムに 2 つの React Native プロジェクトを作成しましたが、両方のプロジェクトの Ruby バージョンを確認すると、両方のプロジェクトの Ruby バージョンが異なり、1 つは 2.7.5、もう 1 つは 2.7.6 です。Ruby バージョン 2.7.5 は、次のいずれかで動作します。私のプロジェクトはエラーなしでした。では、ローカルの Ruby バージョン 2.7.6 を別のプロジェクトの 2.7.5 にダウングレードするにはどうすればよいでしょうか?

プロジェクトから Ruby をアンインストールしようとしましたが、うまくいきません。

解決策

プロジェクトの Ruby バージョンをダウングレードする代わりに、複数の Ruby バージョンをコンピュータにインストールし、プロジェクトごとに正しい Ruby バージョンを選択することをお勧めします。 rbenv などのツールを使用して、複数の Ruby バージョンをインストールおよび管理できます。

チェックアウトするリンクは次のとおりです。 https://translate.google.com/translate?hl=ja&sl=en&tl=ja&u=https://github.com/rbenv/rbenv
