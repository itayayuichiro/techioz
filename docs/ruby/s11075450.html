編集3: CreatePostsの移行(db/merge/##..##_create_posts.rbから)

class CreatePosts < ActiveRecord::Migration
  def change
    create_table :posts do |t|
      t.string :name
      t.string :title
      t.text :content

      t.timestamps
    end
  end
end

解決策

データベースを移行しましたか?

実行してみてください:

heroku run rake db:migrate
