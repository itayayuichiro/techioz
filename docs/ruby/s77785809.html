解決策

Glimmer の説明の状態に注意してください

Ruby 2.6.1 で問題が発生しましたが、Ruby 3.3.0 では次のようにするだけで正常に動作します。

gem install glimmer-dsl-libui

したがって、Ruby 3.x を試す必要があるかもしれません。
