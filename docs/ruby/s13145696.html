Ruby on Rails : 新しいルートを追加する

概要

私は RoR を初めて使用するため、これは初心者の質問です。 コントローラー users_controller.rb があり、メソッド foo を追加した場合、このルートを作成すべきではないでしょうか?

それを実行すると、次のエラーが発生したためです。

もちろんビュー foo.html.erb を追加しました

編集： このコードをroutes.rbに追加しましたが、同じエラーが発生します。

resources :users do
    get "signup"
  end

解決策

これは Rails 3 では自動的に機能しません。追加する必要があります。

resource :users do
    get "foo"
end

あなたのルートへ.rb

http://guides.rubyonrails.org/routing.html をぜひご覧ください。ルーティングについて非常に詳しく説明されています。
