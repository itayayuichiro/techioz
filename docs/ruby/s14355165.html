CocoaPods のインストール: 応答なし

概要

ターミナルから CocoaPods をインストールしようとします:

$ sudo gem install cocoapods

root パスワードを入力しても何も起こりません。

どうすればそれをデバッグできますか?

解決策

同じように疑問に思っている人もいるでしょうが、Gem のインストールには永遠に時間がかかります。実行すると:

export GEM_HOME=~/.gems
export PATH=$GEM_HOME/bin:$PATH

gem install cocoapods -V

フラグ V を指定してインストールすると、詳細な出力が有効になり、ダウンロードとインストールの実行中にすべての出力が表示されます。これは非常に多くの出力です。
