jsonb_accessor と Rails 7.1 を使用した列挙型

概要

jsonb_accessor gemを使用したアプリがあります。

例として、プロジェクトでは次のようなものを使用します。

jsonb_accessor :user_data, gender: :string

enum gender: { male: 'male', female: 'female' }

Rails 7.1 にアップグレードすると (このコミット: https://github.com/rails/rails/commit/6c5fab0668c1872fe827507f45ef400a20e8c646)、エラーが発生します。

ユーザーの不明な列挙属性「性別」 (RuntimeError)

同様の問題を抱えていて、その解決策を見つけた人はいますか?

よろしくお願いします。

敬具、 パブセ

解決策

このスレッドを完了するには:

この機能は PR#49769 で再びレールに追加されます。

事前に型を宣言するだけです。

class Post < ActiveRecord::Base
  attribute :topic, :string
  enum topic: %i[science tech engineering math]
end
