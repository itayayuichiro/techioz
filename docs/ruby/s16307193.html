haml の div title 構文とその中に Ruby 変数も組み込む

概要

HTML 構文を HAML に変換するのに問題があります。

クラスは次のように単純に表現できることはわかっています。

.class-name

ただし、次の構文を HAML に変換し、タイトルに変数も組み込みたいと考えています。

<div title="My title" class="my-text">This is my title</div>

何かのようなもの：

.title{@ruby-variable}.my-text

解決策

.my-text{:title => "My title"} This is my title

haml では #someid.someclass{:custom_attr => “something”} を使用します。 :costum_attr を動的に定義する必要がある場合は、クラスまたは ID にすることもできます。
