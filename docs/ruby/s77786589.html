–css ブートストラップを使用した Rails 7 の新しいアプリ - ターボ ボタンが機能しない

概要

Rails new myapp –css bootstrap を使用して新しいアプリ Rails 7 を作成すると、ターボ ボタンが機能しません。

= button_to “サインアウト”、edit_post_path、メソッド: :delete、フォーム: { data: {turbo_confirm: “よろしいですか?” } }

“本気ですか？”ボタンをクリックしてもポップアップが表示されないのですが、どうすれば修正できますか?

ポップアップはターボでも動作するはずです

解決策

次のものが使用できます。

= button_to "Sign out", edit_post_path, method: :delete,
            data: { turbo_confirm: "Are you sure?" }

edit_post_path についてはよくわかりません。投稿リソースの場合、削除パスは通常 post_path になります。どのようなルートが設定されていますか?
