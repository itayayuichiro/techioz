ランバンドルが完成しました！ 8 個の Gemfile 依存関係、37 個の gem がインストールされました。 バンドルされた gem がインストールされている場所を確認するには、bundle info [gemname] を使用します。不足している宝石をインストールするには

概要

Warvox という Ruby ベースのツールをインストールしようとしていますが、「make」を実行しようとすると、次のエラーが表示されます。

user@localhost:/home/warvox$ sudo make database
Could not find lumberjack-1.0.9 in any of the sources
Run `bundle install` to install missing gems.
make: *** [database] Error 7

そこで私はLumberjackをインストールしました

user@localhost:/home/warvox$ gem install lumberjack
Successfully installed lumberjack-1.0.9
Parsing documentation for lumberjack-1.0.9
Done installing documentation for lumberjack after 0 seconds
1 gem installed

ただし、不足している gem をインストールした後も、エラーは同じままです。アドバイス/提案があれば。

解決策

ステップ 1: バンドラーをインストールする

インターネットに接続されているコンピューターでターミナル ウィンドウを開き、アプリケーション ディレクトリに移動し、コマンド ラインで次のように入力します。

$ gem install bundler

ステップ 2: 必要な Gem をインストールする

Gemfile で指定されたすべての gem をアプリケーションにインストールするようにバンドルに依頼します。

$ bundle install

運用モードで使用されるデータベースとは異なるデータベースを開発モードで使用している場合は、代わりにこれを使用します。

$ bundle install --without production
