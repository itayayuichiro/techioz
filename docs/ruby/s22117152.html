Ruby on Rails:-Rails と Rails Server コマンドはすべて新しいプロジェクトを作成します

概要

いくつかの奇妙なレールの動作が発生しています。

次のコマンドを実行すると

rails new blog

rails s

rails server

それぞれの結果は新しいプロジェクトです。つまり、new、s、server という名前の 3 つのフォルダーがあり、その中に新しい Rails プロジェクトが含まれています。

…なぜこうなった？？使用しているバージョンに関係があるのではないかと感じています。rvm を使用して Ruby を 1.87 から 2.0 に更新し、Rails 2.3.14 をインストールしました。

私はxubuntuを使用していますが、これもUnityから切り替えたばかりです。

解決策

Rails 2.3 アプリケーションで Rails 3.x コマンドを使用しようとしています。 3.0 より前のアプリケーションの場合は、以下で説明するサーバー スクリプトを使用する必要があります。

アプリケーション ディレクトリ内から次を実行します。

./script/server
