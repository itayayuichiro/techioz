Rails の移行: ROR 移行を使用して列のデータ型のサイズを増やす方法

概要

私のユーザーテーブルのログイン列は文字列型で、制限は 40 文字です。 現在、制限を 55 文字に増やす予定です。

ROR移行を使用してこの制限を増やす方法を教えてください。

ありがとう、 スラヴァン

解決策

class YourMigration < ActiveRecord::Migration
  def up
    change_column :users, :login, :string, :limit => 55
  end

  def down
    change_column :users, :login, :string, :limit => 40
  end
end
