Linuxのスクリプトを使用して付箋に書かれたテキストを読むことはできますか?

概要

私はubuntuで付箋を使用しています。そして、付箋に書かれたテキストをスクリプト言語を使用して読むことができるかどうか疑問に思っていました。

解決策

パネルに追加できる「Sticky Notes」アプレットのことを指していたのであれば、はい、そのメモを読むこともできます。

すべてのメモを含む XML ファイルは通常、~/.gnome2/stickynotes_applet にあります。

そこから必要な情報を解析するだけで済みます。構造は次のようになります。

<?xml version="1.0"?>
<stickynotes version="2.30.0">
    <note title="10/31/2010" x="658" y="176" w="477" h="418">Some text</note>
</stickynotes>

ここで、x は x 軸上のノートの位置を表し、y は y 軸上のノートの位置を表し、w は幅を表し、h は高さを表します。

たとえば Perl を使用してパーサーを構築するのは非常に簡単です。
