送信cliの終了方法

概要

Ruby スクリプトを使用して bitcoin bootstrap.dat ファイルをダウンロードしています。このファイルは torrent クライアントを使用することによってのみダウンロードできるため、コマンドラインでダウンロードするには、次の Ruby デモ スクリプトを使用しています。

#!/usr/bin/ruby env
`transmission-cli https://bitcoin.org/bin/blockchain/bootstrap.dat.torrent`

今私はそのスクリプトを実行します

root@master:~# ruby demo

しかし、上記の送信では torrent ファイルが完全にダウンロードされますが、その後シードとアップロードが開始され、自動的に終了しないため、スクリプトがハングします。スクリプトを使用してこれを終了する方法はありますか?

トレントのダウンロードが完了した後にキルまたは終了シグナルを送信できる方法はありますか?

解決策

ダウンロードの完了時にスクリプトを実行する -f, -finish
