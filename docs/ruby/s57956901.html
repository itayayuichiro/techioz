YouTube RSSフィードの解析

概要

Rubyのようにこれらのメディアキーを読み取るための高速でクリーンなサンプルはありますか?

すでにRuby RSSを使用していて、タイトル、リンクを読んでいますが、セミコロンが付いたこれらのメディアキーを読むことができません。ここで nokogiri と xml key read を使用する必要がありますか?それとも標準の Ruby RSS ライブラリにこのキー用の何かがあるのでしょうか?

YouTube チャンネル フィードを読み取るために使用しているコード:

require 'rss'

rss = RSS::Parser.parse( "https://www.youtube.com/feeds/videos.xml?channel_id=UCAUZpKBrZ-QP72Y0blet6DA", false)

rss.items.take(2).reverse.each do |item|
puts item.title.content
end

解決策

RSS フィードは XML を返す必要があります。そのため、ActiveSupport gem を使用して解析できます。

require 'rss'
require 'active_support/core_ext/hash'

rss = RSS::Parser.parse( "https://www.youtube.com/feeds/videos.xml?channel_id=UCAUZpKBrZ-QP72Y0blet6DA", false)

array = rss.entries.map do |e|
  Hash.from_xml(e.to_s)
end

# here is exactly what is contained in this array:
array.each{|i| puts i.inspect}
