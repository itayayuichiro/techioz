「SDK が指定されていません」という名前の Ruby コードを実行するとエラーが発生する

概要

Ruby コードを実行するために RubyMine IDE をダウンロードしました。次のコードを書きました。

puts 'hello world'

しかし、この最小のプログラムを実行しようとするとエラーが発生します。 IDE に次のエラーが表示されます。

"No SDKs specified"

なぜ ？この問題を解決したい場合はどうすればよいですか?私はこの言語の初心者です。この点について手伝ってもらえますか？また、あなたの提案も欲しいです。「RubyMine」は Ruby に最適な IDE ですか?どのようなIDEを使用していますか?

解決策

ここで SDK を設定する必要があります。

[SDK の追加…] ボタンを使用して、システム上の Ruby 実行可能ファイルへのパスを指定します。

Windows では、http://rubyinstaller.org/ バイナリ リリースを使用して Ruby をインストールする必要があります。ソースパッケージではありません。
