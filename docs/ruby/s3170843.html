緑色の糸とは何ですか？

概要

Ruby on Rails のグリーン スレッドとは何を意味しますか?実際に何をするのですか? あれですか？ 「固定サイズのスレッドプールがプログラム全体で共有される設計ですか?」 b. 「スレッドが仮想マシンまたはインタープリタによってエミュレートされるときは?」または c. 「複数の CPU にわたってプログラムが実行される場所はどこですか?」

正しい答えで説明してください?

解決策

グリーン スレッドは、オペレーティング システムではなく仮想マシンまたはランタイム ライブラリによってエミュレートされるスレッドです。
