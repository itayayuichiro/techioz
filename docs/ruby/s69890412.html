概要

gem "json", "~> 2.6"

いくつかの解決策（バンドルとjekyllの更新と再インストール、webrickとjson gemの追加など）を試しましたが、役に立ちませんでした。これは新しいマシンなので、何かが間違って設定されているはずですが、それが何かはわかりません。

解決策

github ページをセットアップしようとして問題に遭遇したときに、この質問に遭遇しました。 homebrew webrick でインストールされる Ruby の最新バージョンにはデフォルトで含まれていないようです。 github の問題は次のとおりです: https://github.com/github/pages-gem/issues/752。 Ruby 2.7 を使用するか、以下のコマンドを使用して Webrick を手動で追加することをお勧めします。

bundle add webrick

そしてコマンドを繰り返します

bundle exec jekyll serve
