そのようなファイルまたはディレクトリはありません - git config init.defaultbranch (Errno::ENOENT)

概要

cmd で Ruby on Rails をインストールしましたが、「No such file or directory - git config init.defaultbranch (Errno::ENOENT)」というエラーが発生しました。この問題は解決できません。

Webガイドを参考にインストールしてみましたが直りませんでした

何か解決策を提案してください。

解決策

ローカルマシンに git をインストールしてこのエラーを修正すると、エラーは消えました。git インストーラー https://git-scm.com/downloads へのリンクを添付してください。
