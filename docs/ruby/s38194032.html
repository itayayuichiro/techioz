Mac OS X v10.10 (Yosemite) で Ruby バージョン 2.0.0 を最新バージョンにアップデートするにはどうすればよいですか?

概要

Ruby のバージョンを 2.0.0 から最新バージョンに更新する必要があります。バージョンが更新されていないため、一部の gem が使用できません。 私は少し前に Homebrew を使用して Ruby をインストールしたことがありました。 Ruby のバージョンを更新するにはどうすればよいですか?

解決策

ターミナルを開いて実行します

curl -sSL https://raw.githubusercontent.com/rvm/rvm/master/binscripts/rvm-installer | bash -s stable

RVM コマンドを機能させるには、以下を実行する必要があります。

source ~/.rvm/scripts/rvm

次に、rvm list known を実行します

Rubyインタプリタのバージョン一覧を表示します。

次に、rvm install Ruby@latest を実行して、最新の Ruby バージョンを取得します。

ターミナルで「ruby -v」と入力すると、「ruby X.X.X」が表示されるはずです。

それでも Ruby 2.0 が表示される場合は、rvm use Ruby-X.X.X –default を実行します。
