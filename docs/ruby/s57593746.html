Mac OS Catalina に fastlane をインストールできない

概要

fastlane のドキュメントに記載されている手順に従って、fastlane をインストールできません。

brew cask install fastlane を実行すると、次のポップアップが表示されます。

sudo gem install fastlane を使用して gem から実行しようとすると、次のエラーが発生します。

ディレクトリにruby.hファイルが存在しないことがわかります。エラーにあるように、ruby-dev 環境を別途インストールする必要があります。しかし、その方法が見つかりませんでした。このための apt-get コマンドは見ましたが、では、brew 経由でインストールされた Ruby はどうなるのでしょうか?

ルビー -v: ルビー 2.6.3p62 brew -v: 自作 2.1.10 brew upgrade Ruby: Ruby 2.6.3 はすでにインストールされています

解決策

開発用に Ruby を別のパッケージとしてインストールする必要があるというエラーが表示されたため、ROR の Web サイトに示されているように rbenv をインストールする必要がありました。完全な問題と解決策はここで確認できます。
