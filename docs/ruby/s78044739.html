スプロケットが適切に動作してアセット パイプラインをインポートしたいと考えています。プロジェクトはすでに稼働しているため、既存のスタイル コードを変更することはできません。アセット パイプラインが適切に動作するように、Sprocket を適切にセットアップして構成する必要があります。

解決策

Rails 7.1のベストプラクティスに従ってJavaScriptをリファクタリングします

それでも Rails 7.1 で jQuery を使用したい場合は、assets フォルダーの下の JavaScript に jQuery を配置し、それを application.js (またはあなたの場合はコーヒー) に含めて、それがフェッチされるかどうかを確認してみます。

また、jquery-rails gem にはいくつかの注意事項があることに気付きました。

デバッグ目的で jquery-ujs を Rails-ujs に置き換え、その後は前述したように jQuery1.17 を配置することをお勧めします。

https://github.com/rails/sprockets-rails もご覧ください。
