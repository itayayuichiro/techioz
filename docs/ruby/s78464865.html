VSCode の Ruby、ワークスペースが見つかりませんでした [終了]

概要

VSCode で Ruby を使用する方法を学び始めましたが、これまで VSCode を使用したことがなく、セットアップに苦労していると感じています コードをデバッグしようとすると、必ず「URI のワークスペースが見つかりませんでした: file:///c%3A/Users/Admin/Desktop/Ruby/RubyWS またはエディタ: [オブジェクト オブジェクト]」というエラーが発生します。 誰か助けてくれませんか、ありがとう！

VSCodeの正しい使い方を学びましょう！

解決策

この記事では、vscode で Ruby コードを実行する方法を学びます。

Ruby バージョン – 3.0.0

Gem バージョン – 3.2.3

これらのパッケージをインストールしたと思います

これらのパッケージをインストールしたと思います

次に、次のコマンドを記述します。

バージョンを指定するよう求められます。

PS C:\Users\sharu\Documents\abcd> gem -v 

3.4.10 

PS C:\Users\sharu\Documents\abcd> ruby -v 

ruby 3.2.2 (2023-03-30 revision e52012f9c0) [x64-mingw-ucrt]

ファイル main.rb に Ruby コードがあると仮定します。

puts 'hello ruby!'

その後、ターミナルで次の操作を行うことができます。

PS C:\Users\sharu\Documents\abcd> ruby main.rb 

hello ruby!

これでコードを実行できるようになります。
