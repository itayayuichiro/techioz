初期化されていない定数 RSpec::Matchers::Composable

概要

非常に多くのものをロードした後、’: uninitialized constant RSpec::Matchers::Composable (NameError) というテスト ケースの実行で問題が発生しました。助けてください。

前もって感謝します！

解決策

Rails アプリのテストケースでもこれと同じ問題に直面していました。私がそれをどうやって解決したか。

follow below two steps:


 1.  first mark the gem as required false  as below
  group 'test' do
     gem 'rspec-sidekiq', require: false
  end
 2. require the gem in rails_helper.rb file as below
     require 'rspec-sidekiq'

今問題は解決されるでしょう
