サーバー起動時に「Ruby のバージョンは 2.3.0 ですが、Gemfile は 2.2.5 を指定しました」を修正する方法

概要

サーバーの実行中にこのエラーが発生します。これを修正するにはどうすればよいですか?

解決策

互換性のために Ruby 2.2.5 をインストールすることをお勧めします。ローカル マシンの Ruby のバージョンは、Gemfile で宣言されたものとは異なります。

RVM を使用している場合:

rvm install 2.2.5
rvm use 2.2.5

rbenv を使用している場合:

rbenv install 2.2.5
rbenv local 2.2.5

rbenv で Ruby のバージョンを変更できない場合は、 ここを読んでください
