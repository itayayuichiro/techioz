RubyOnRails + Mac M1 + Rails サーバーの実行中に、YAML 出力の psych が見つからないというエラーが発生しました

概要

Mac M1 を使用し、Ruby Ruby 2.5.3p105 (2018-10-18 リビジョン 65156) [-darwin23] と Rails 5.2 を実行していますが、このエラーが繰り返し発生し、いかなる開発も行うことができなくなります。 Railsで。

Rails の実行中に、コンソールに次のエラーが表示されました。

libyamlをインストールして再リンクしようとしました。まだ動かない。

RVM経由でRubyの再インストールが完了しました。

解決策

Ruby 2.5.3 を実行しているときに M2 Mac でうまくいったこと
