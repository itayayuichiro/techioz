OSが所有するRubyファイルを更新できません

概要

sudo gem update –system を使用して ubuntu で Ruby を更新しようとしていますが、エラーが発生し続けます。

gemsライブラリ全体をchownしましたが、役に立ちませんでした。私に何ができる？

解決策

これは、古い Ruby バージョンが作成された場合に発生する可能性があります。 .rb ファイル。通常、それは次の場所にあります

/usr/lib/ruby/vendor_ruby/rubygems/defaults/operating_system.rb

削除して実行しようとすると

gem update --system

私のために働いてくれました。
