バンドル パックとバンドル キャッシュの違いは何ですか

概要

最新バージョンのバンドラー ドキュメントにはバンドル パック コマンドが表示されませんが、古いバージョンのままです。非推奨であることについても言及されていません（私の知る限り）。ドキュメントの内容はバンドル キャッシュに似ています。それらは同じであると考えても問題ありませんか?

解決策

Bundle Pack は、bundle package コマンドのエイリアスです (またはエイリアスでした)。

バンドル キャッシュとバンドル パッケージは、以前は別個のコマンドでした。 Bundler 2.4.0 では、以前の package サブコマンドがプル リクエスト #7724 および #7389 のキャッシュ サブコマンドに統合されました。

3 つのコマンド (cache、package、pack) はすべてまったく同じことを実行し、単純なエイリアスになりました。サブコマンドの主な名前としてバンドル キャッシュを使用することをお勧めします。
