エラー: gem の実行中 … (Gem::Exception) OpenSSL は使用できません

概要

brew install openssl を通じて openSSL をインストールしているにもかかわらず、gem をインストール/更新しようとすると、この openssl エラーが発生します。

これはイライラするかもしれませんが、この問題に対処する方法があります。

解決策

この問題に対処するには、次のことを行う必要があります。

RVM の場合、これは次のことを意味します。

rvm reinstall 2.3.0 --with-openssl-dir=`brew --prefix openssl`

フラムの場合:

frum uninstall <ruby version>
frum install <ruby version> --with-openssl-dir=<ssl_dir> # Specify the OpenSSL directory

このガイドは MacOS を対象としています。同様に、使用しているオペレーティング システムに適切な cli オプションを指定して Ruby を再インストールする必要があります。
