Ruby float を最も近い 0.05 に四捨五入します。

概要

次のような数字が得られます

2.36363636363636
4.567563
1.234566465448465
10.5857447736

Ruby にこれらの数値を最も近い 0.05 に切り上げ (または切り捨て) させるにはどうすればよいでしょうか?

解決策

[2.36363636363636, 4.567563, 1.23456646544846, 10.5857447736].map do |x|
  (x*20).round / 20.0
end
#=> [2.35, 4.55, 1.25, 10.6]
