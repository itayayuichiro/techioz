Jekyll が Ruby 3.1 でまったく動作しないのはなぜですか?

概要

Ruby 3.1を使用して実行しました。

gem install jekyll

それから私は実行しました:

jekyll -v

そして、私はこのエラーを受け取りました:

`require': cannot load such file -- sass-embedded (LoadError)

Ruby 3.1.2 を Ruby-3.2.0-preview1 にアップデートしようとしましたが、さらにエラーが発生しました。

解決策

Sass-embedded をインストールしました。

gem install sass-embedded

今はうまく機能しているようです。
