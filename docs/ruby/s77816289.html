Rubyでオブジェクトを初期化できない

概要

Rubyでオブジェクトを初期化できない

しかし、それが初期化メソッドの目的ではないでしょうか?

class Thing
  def initialize(id, color)
    @id = id
    @color = color
  end
end

Thing thing = Thing.new(1, "blue")

解決策

ローカル変数は、単に代入するだけで作成されます。それらを宣言したり、その型を指定したりする必要はありません。 (実際にはできません)

書くだけ：

thing = Thing.new(1, "blue")
