LoadError: dlopen(/Library/Ruby/Gems/2.6.0/gems/nokogiri-1.12.5-x86_64-darwin/lib/nokogiri/2.6/nokogiri.bundle, 0x0009): could not use '/Library/Ruby/Gems/2.6.0/gems/nokogiri-1.12.5-x86_64-darwin/lib/nokogiri/2.6/nokogiri.bundle' because it is not a compatible arch - /Library/Ruby/Gems/2.6.0/gems/nokogiri-1.12.5-x86_64-darwin/lib/nokogiri/2.6/nokogiri.bundle

私は Rbenv を使用しています。これには正しいパスがあり、ドクター スクリプトから OK が得られます。

ターミナルで次を実行すると、次の結果が得られます。 Ruby -v Ruby 3.0.0p0 (2020-12-25 リビジョン 95aff21468) [arm64-darwin20] レール -v レール 6.1.4.1 /usr/local/bin/rails をレールにする

Mac にはデフォルトの Ruby バージョンが付属しており、Rails New 中にこれを使用しようとしているようです。これはわかりません。

解決策

x86プロセッサ用のUsing nokogiri 1.12.5 (x86_64-darwin)で試していますが、arm64-darwinバージョンが必要です。試すための M1 マシンはありませんが、次の方法で Nokogiri を再インストールしてみるとよいでしょう。

gem uninstall nokogiri
bundle config set force_ruby_platform true
bundle install
