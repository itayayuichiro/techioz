Flutter デバッグ: 警告: CocoaPods がインストールされていません。ポッドのインストールをスキップする

概要

私の問題はこれです:

アプリを Firebase に接続したいと考えています。すべての手順に従い、指示に従ってすべてを実行しました。アプリをデバッグしようとすると、次のエラーが発生しました。

ターミナルから - flutter run - と入力すると、次のことが起こります。

Flutter Doctor -v では次のようになります。

そして、次のバージョンとパスがあります。

この問題をどうやって解決すればよいか知っていますか?

PS: 私は Visual Studio Code と XCode に取り組んでいます。 MacOS Big Sur バージョン 11.2 を使用しています。

解決策

私も過去に同じエラーが何度か発生しており、Flutter 2.0.1 にアップグレードした直後にも一度発生しました。私にとっての解決策は、(以前に提案したように) 常に実行することでした。 sudo gem install Cocoapods してから IDE を再起動します。私は Android Studio を使用していますが、VSCode にも同じことが当てはまります。
