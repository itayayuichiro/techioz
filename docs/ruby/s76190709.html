Debian 11、Mastodon サーバー、Ruby のインストールの問題、ユーザーの Mastodon

概要

[Mastodon のインストール] プロセスを説明した多くのマニュアル (https://docs.joinmastodon.org/admin/install/) には、次のように書かれています。 “rbenv は 1 人の Linux ユーザーに対してインストールする必要があるため、最初に Mastodon が次のように実行されるユーザーを作成する必要があります。 adduser –disabled-login マストドン 次に、ユーザーに切り替えることができます。 す、マストドン」

Debian 11 で「–disabled-login」アカウントでログオンすることはできますか?私にはそれができません。

コマンド su - マストドン を使用してみます。 他のユーザー アカウントでは、システムがパスワードを要求します。 パスワード（ユーザーまたはrootのパスワード）を書き込むと、次のように表示されます。 「su：認証失敗」 root アカウントでは次のように表示されます。 「このアカウントは現在利用できません。」

解決策

同じ問題が発生しました。代わりに sudo -u mastodon bash を使用できます。

出典: https://github.com/NixOS/nixpkgs/issues/199029#issuecomment-1301964553
