asdf を使用して Ruby バージョンを 3.2.2 から 3.3.0 にアップグレードするにはどうすればよいですか?

概要

私は現在、asdf を使用してインストールした Ubuntu 22.04 で Ruby バージョン 3.2.2 を使用しています。バージョン 3.3.0 にアップグレードしたいと考えています。

asdf install Ruby 3.3.0 を実行すると、「ruby-build: 定義が見つかりません: 3.3.0」というエラー メッセージが表示され続けます。

asdf を使用して Ruby のバージョンを 3.3.0 にアップグレードする方法を誰かが教えてくれませんか?

asdf install Ruby 3.3.0 を実行してみましたが、バージョンが Ruby 3.3.0 にアップグレードされると予想されました。

解決策

次のコマンド asdf plugin update Ruby を使用して asdf プラグインを更新すると、うまくいきました。 その後、3.3.0 をインストールできるようになります。
