Selenium Webdriver Chrome Webdriver インスタンスを作成できません

概要

Google Chrome と Firefox の両方で Selenium を使用して、Ruby on Rails 2 に対する統合テストを正常に実行してきました。ただし、最近 Ruby on Rails 3 にアップグレードしたところ、Google Chrome Webdriver インスタンスの作成で問題が発生しました。

任意のヒント？

解決策

これは、依存関係の一部としてバックポート gem を含む gem を Gemfile に含めたときに発生しました。バックポート gem は Ruby 1.8 の内部の一部を書き換え、UDPSocket コア クラスの一部を上書きします。バックポート gem を削除して試してみてください。
