Rails コンソールがループの最後にオブジェクトを出力しないようにする

概要

たとえば、指定されたモデルのすべてのインスタンスをループして、それぞれから何かを出力したとしても、最後に irb はオブジェクト全体を出力します。

オブジェクトが最終的に何百行もかかる場合、実際に探していたものを見つけるまでには長い道のりがかかります。 Railsコンソールでこれを無効にする方法はありますか?

解決策

通常、エコーを無効にしたくない場合は、1 つのコマンド ラインで複数の式を呼び出すこともできます。最後の式の出力のみが表示されます。

big_result(input); 0
