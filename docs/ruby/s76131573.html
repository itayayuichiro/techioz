Bright Data スクレイピング ブラウザ: 「禁止されたアクション: パスワードの入力は許可されていません」

概要

Playwright Python ライブラリ経由で Bright Data スクレイピング ブラウザを使用して、次のコード スニペットを作成しました。

await page.goto('https://example.com/login/')
await page.fill('input[name="username"]', username)
await page.fill('input[name="password"]', password)
await page.click('button[type="submit"]')

ユーザー名は期待どおりに入力されますが、パスワード行では、スクレイピング ブラウザが次のエラーで停止します。 エラー: 禁止されたアクション: パスワードの入力は許可されていません

入力にデータを入力する他の方法を試しましたが、スクレイピングブラウザではパスワードタイプの入力フィールドへのあらゆる種類のデータの入力が無効になっているようです。

これを回避するにはどうすればよいですか?

解決策

Brightdata はセキュリティ上の理由から、Brightdata 側でのパスワード入力をブロックしているため、同様の許可を得る必要があります。 チケットを開いて目的を述べることで、彼らにアプローチできます。
