特定のバージョンの Ruby-dev を termux にインストールする方法

概要

Android上のtermux上にRuby on Rails環境をセットアップしようとしています。 2.1.10 Rubyを使いたいと思っています。私が見つけたのは、最新バージョンの Ruby をインストールする apt install Ruby-dev または pkg install Ruby-dev だけでした。助言がありますか？

解決策

RVMをインストールすることができます。 0. 推奨事項: tmux と zsh をインストールし、zsh をデフォルトのシェルとして変更します。ああ、Zsh は機能しました。 Powerline フォントも (termux スタイルのスクリプトをインストールすることで)

RVM をインストールしましょう。

 case "`uname` in
(CYGWIN*|MINGW*)
... etc ...

3 つの初期化変数 __shell_name 文字列を次のように置き換えるだけです。

__shell_name="gawk 'BEGIN{RS=\"\"}; NR==1{print; exit
}' /proc/$$/cmdline | tr - '\0'" ;;

termux で動作する RVM: https://github.com/rvm/rvm/issues/4734
