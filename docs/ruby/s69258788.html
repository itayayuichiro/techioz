JSONAPI.org のようなネストされたハッシュ構造で require を使用することはできますが、これは単一のキーを返すだけで、最初に欠落しているキーで発生するため、入力の何が間違っているかについて意味のある詳細なフィードバックを提供するために使用することはできません。

概要

解決策

今後このスレッドをチェックする人のために、ruby 3.0.0 では rexml がデフォルトからバンドルされた gem に移動されました。 Ruby 3 にアップデートする前に aws-sdk-core が機能していた場合は、rexml を gemfile に追加するだけです。 もちろん、エラーメッセージで述べたように、以前の回答は機能します。どの XML ライブラリでも問題ありません。
