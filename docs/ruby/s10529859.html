ジキル マークダウン ブログにビデオを含める方法 [終了]

概要

jekyllを使ってブログを始めました。私はマークダウンで投稿を書きます。ここで、投稿に YouTube ビデオを含めたいと思います。これどうやってするの？

私は、jekyll がデフォルトで提供するピグメントのハイライトがあまり好きではありません。とにかくこれを他のスタイルに変更することはできますか? 「はい」の場合、いくつかの素敵なスタイル/プラグインを教えていただけますか?

解決策

埋め込み用の HTML をマークダウンに直接挿入できるはずです。ビデオの下に [共有] ボタンがあるので、これをクリックしてから [埋め込み] ボタンをクリックすると、次のようなものが表示されます。

<iframe width="420" height="315" src="http://www.youtube.com/embed/dQw4w9WgXcQ" frameborder="0" allowfullscreen></iframe>

それをコピーして投稿に貼り付けるだけで、Markdown プリプロセッサはそれに影響しません。

Pygments の場合、このリポジトリにはさまざまなカラー テーマの CSS スタイルシートが山ほどあり、試してみることができます。 (これらを Jekyll で動作させるには、.codehilite を .highlight に置き換える必要があることに注意してください。)
