Unity が iOS リゾルバーを実行し続けて Cocoa ポッドをインストールしようとする

概要

Unity は、Cocoapod をインストールするために iOS リゾルバーを実行しようとし続けます。

ターミナルでは以下のことを確認しました。以下も再インストールしました。

同じリポジトリを使用している私の友人も、彼の側では問題を経験していません。彼が実行している Ruby のバージョンは 2.6.10.210 です。

コードまたはフォルダー階層に書き込まれていると理解していますが、Unity のどこを参照または編集すればよいかわかりません。 chatgpt と Google bard を使用してみましたが、うまくいきませんでした。

私もMacを使うのは初めてです。

Unity バージョン 2022.1.18f1 を実行しています

解決策

Unity は /usr/local/bin フォルダーで Cocoapod を検索し、次に /Users/theyashiverse/.gem/ruby/2.6.0/bin で Cocoapod を検索します。

まず、既存の Cocoapods gem を削除します。

gem uninstall cocoapods

ターミナルから /usr/local/bin に Cocoapod をインストールしてみます

gem install -n /usr/local/bin cocoapods
pod setup

次に、Unity を再起動して、もう一度試してください。
