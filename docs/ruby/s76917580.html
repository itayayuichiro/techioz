microsoft_graph Ruby on Rails SDk

概要

Ruby on Rails プロジェクトで Microsoft_graph API を使用しようとしていますが、エラーが発生します Errno::ENOENT: No such file or directory @ rb_sysopen - C:/Ruby32-x64/lib/ruby/gems/3.2.0/gems/microsoft_graph -0.22.1/lib/communications/call_records/microsoft_graph_call_records_get_direct_routing_calls_with_from_date_time_with_to_date_time/get_direct_routing_calls_with_from_date_time_with_to_date_time_response.rb gem バンドル gem「microsoft_graph」をインストールしようとしたとき。 Ruby on Rails は他の言語と同様に Microsoft のドキュメントでコミットされているため、Ruby on Rails ではサポートされていないと私は本気で思っています。誰かがそれを使用したか、ドキュメントを教えてくれたら知りたいです

解決策

現在のこの SDK のドキュメントによると: 注: Microsoft Graph Ruby SDK は現在コミュニティ プレビュー段階にあります。この期間中に、コミュニティからのフィードバックに基づいて SDK に重大な変更が行われることが予想されます。既知の制限事項を確認してください。

とはいえ、今日、Ruby 3.0.0と最新のrailsを使用してこのSDKをrailsアプリにインストールしました。 - V7.0

SDK には、Github の問題で確認できる問題がいくつかあります。インストールに関しては、最新のRubyであればSDKをインストールできるはずです。
