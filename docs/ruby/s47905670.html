Redmine 3.3 に AgileDwarf プラグインをインストールするときの初期化されていない定数 ApplicationHelper::GravatarHelper

概要

私は git clone https://github.com/iRessources/AgileDwarf から /usr/share/redmine/lib/plugins にプラグインをクローンしています。ディレクトリ /usr/share/redmine で実行すると、bundle install –withoutdevelopment test –no-deployment 出力:

次のバンドルは rake redmine:plugins NAME=AgileDwarf RAILS_ENV=production を実行します。出力

Apache2を再起動するとき。そしてサイトには次のようなエラーが書かれていました。

Redmineのバージョン：

解決策

AgileDwarf プラグインは約 5 年間メンテナンスされていないようです。プロジェクトに記入されたさまざまな問題から判断すると、プラグインは Redmine 3.x と互換性がないようです。プラグインのすべての層に複数の問題があり、現在の Redmine バージョンでの使用を妨げているようです。

Redmine 3.x のサポートを提供しているプラグインのフォークが見つかるかもしれません。

あるいは (これがおそらく今後のより良い選択肢です)、アジャイルな問題管理のための別のプラグインを見つけることもできます。 RedmineUPのアジャイルプラグイン。
