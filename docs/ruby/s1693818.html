# Rails の TODO

概要

Rails の本を読んだところ、コードに # TODO: や # ものを追加できることがわかりました。そのため、cmd をレイクして振り返ることができます。

私の問題は、Google で検索しても「# スタッフ」や「rake cmd」がどこにあるのか見つからず、どのキーワードで検索すればよいのかもわかりません。

解決策

TODO、# FIXME、# OPTIMIZE を使用でき、次の rake タスクを使用してこれらすべてを簡単に取得できます: rake Notes

デフォルトの注釈に加えて、いつでも独自の注釈を指定して検索できます。

rake notes:custom ANNOTATION=ZOMG
