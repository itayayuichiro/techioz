解決策

OSX Montrerey と Ruby 2.7.0 のインストールで問題が発生した場合は、無駄な努力をせずに次の方法を使用してください。

ソリューションの後半のソースはここで、OS によってインストールしたばかりのバージョンを選択します https://translate.google.com/translate?hl=ja&sl=en&tl=ja&u=https://stackoverflow.com/a/39238995/3086197

2.7.0 を必要なバージョンに置き換えます
