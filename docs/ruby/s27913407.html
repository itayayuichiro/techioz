EBCDIC から RUBY の ASCII へ

概要

メインフレームから生成された EBCDIC ファイルがあるため、データ処理のためにそれを ASCII に変換する必要があります。 助けていただければ幸いです。

解決策

[Ruby 2.3 以降、EBCDIC エンコーディングが利用可能][1]:

したがって、これは機能するはずです:

src = 'out_26877296.tst'
content = File.read(src, encoding: 'IBM037:ASCII')
