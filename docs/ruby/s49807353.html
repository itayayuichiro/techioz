解決策

マシンに Mysql がインストールされていないか、アダプターの設定が間違っているようです。

まず、mysql サーバーをマシンにインストールします。 その後、以下のコマンドまたはアプリケーション gemfile に追加して mysql2 gem をインストールします

gem インストール mysql2

宝石「mysql2」

次に、アダプターに対して以下の設定を行います

デフォルト:&デフォルト

アダプター: mysql2

ユーザー名：xxxxx

パスワード：xxxxx

ホスト: ローカルホスト

プール: <%= ENV.fetch(“RAILS_MAX_THREADS”) { 5 } %>

タイムアウト: 5000

これがお役に立てば幸いです。
