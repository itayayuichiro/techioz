Ruby の別のバージョンでインストールしようとしましたが、うまくいきませんでした。

解決策

私もこの問題を抱えていましたが、最終的に解決したのは次のとおりです。

gem install mysql2 -v ‘0.5.3’ – –with-opt-dir=/usr/local/opt/openssl@3 –with-ldflags=-L/opt/homebrew/Cellar/zstd/1.5.0 /lib

ただし、価値があるものとして、以前はインストールできませんでしたが、次のことも必要だった可能性があります。
