解決策

コードの一部でまだファラデーマルチパート gem が必要のようですが、Gemfile でコメントアウトされています。 コメントを外し、bundle を再度実行して、サーバーを再起動してみてください。

さらに、古いバージョンの google-api-client を使用しているようです。問題はその gem に由来しており、faraday/request/multipart ファイルを要求しようとしますが、そのファイルは新しいバージョンには存在しません。ファラデーマルチパート宝石。

Google API gem を更新してみてください。まず google_drive から始めます。
