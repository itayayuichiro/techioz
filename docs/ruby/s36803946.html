フードトラック用の Rails アプリ。twitter を使用して Google マップを更新します。

概要

そこで私は友人のフードトラックビジネスのために Rails アプリケーションを書いています。

この特定のアプリケーションで作成しようとしている機能は、Twitter に地理位置情報タグを投稿することで Google マップを更新する機能です。

Google マップを扱う Ruby gem の多くは価値が下がっているようで、現時点では少し迷っています。誰かが私に正しい方向を教えてくれれば、感謝します。

この機能が単に生活の質を向上させるための機能であることは理解していますが、JavaScript を中断する必要がなく、埋め込み Ruby で実行できるようにしたいと考えています (うーん)。私の友人は移動中に常に Web サイトのバックエンドにアクセスできるとは限らないため、この種の自動化はこの特定のタイプの Web サイトに最適であることが理想です。 (管理バックエンドから地図を更新することはすでに可能ですが、1 日に複数の場所を移動するビジネスには十分ではありません。)

とにかく、よろしくお願いします。 (私のコードを投稿してほしい人がいたら、投稿します。)

解決策

それで、あなたの質問を読んだ後、しばらくTwitter APIを使っていなかったので、もう一度Twitter APIをいじってみるのは楽しいだろうと思いました。

私は、基本的にあなたが求めていたことを実行するサンプル アプリケーションを作成しました。ユーザーの最近のツイートを検索して最新のツイートを取得し、その場所にマーカーを付けて Google マップを更新します。

これは急いでまとめたものなので、完璧とは程遠いことに注意してください。しかし、すべての概念がそこにあるはずです。
