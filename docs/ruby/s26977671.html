Rubyで関数を呼び出す方法

概要

タスクから install_apache 関数を呼び出すのが困難です。

task :install_apache => :environment do
  begin
    install_apache
  rescue Exception => e
    puts e
  end
end

def install_apache
  # Code...
end

解決策

Ruby では、最初にメソッドを定義する必要があります。構文を使用してそれを行います

def method_name
  # code goes here
end

メソッドを呼び出すには、メソッド名を入力するだけです

method_name

あなたの場合、最初のステップダウンがありました

def install_apache
end

次のようにコードに別の行を追加する必要があります

install_apache
