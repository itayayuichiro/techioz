Windows で Fluentd のインストールが失敗し、メッセージが表示される

概要

Windows OS で Apache Tomcat からログ ファイルを収集するために Fluentd のテストを行っています。 そこで、[Fluentd-Packages v5.x] と [Calyptia-Fluent v1.3.x] でインストールしようとしましたが、以下のような「セットアップ ウィザードが途中で終了しました」という Windows メッセージが表示されてインストールに失敗しました。

5 つの Windows ボックスで試しましたが、1 つの Windows ボックスでのみ成功しました。異なるのは、Windows Update とインストールされた .NET Framework のみです

1. Succeed Windows : didn't update from 1st installation
2. Other Windows : did update all windows

結局、解決の糸口は見つかりませんでした。 Windows と Fluentd、Ruby の専門家の助けが必要です。知識がある場合は、私に共有してください。 ありがとう。

解決策

私もこれと同じ問題に遭遇しました - 別のインストールが失敗した後に残った fluentd サービスを削除することで解決しました

sc delete fluentdwinsvc
