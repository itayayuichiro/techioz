解決策

レール 7

./bin/importmap pin aos

インポートマップ.rb

pin 'aos', to: 'aos.js', preload: true

=> ベンダー/javascript/aos.js

アプリケーション.js

import AOS from 'aos';
document.addEventListener('turbo:load', () => { AOS.init() });

構成/アセット

Rails.application.config.assets.precompile += %w[aos.js]

私にとっては役に立ちます。
